简介

html：超文本标记语言，是一种用于创建网页的标准标记语言。文档后缀为 .html 或 .html 。

html 编辑器选择：1.VS Code；2.Sublime Text；3.source insight；4.Notepad++ 等。

注：在浏览器的页面上使用键盘上的 F12 按键开启调试模式。

标签：

  1. HTML 标签是由尖括号包围的关键词，比如 <html>；
  2. HTML 标签通常是成对出现的，比如 <b>...</b>；
  3. 标签对中的第一个标签是开始标签，第二个标签是结束标签；
  4. 开始和结束标签也被称为开放标签和闭合标签。

HTML 元素：

  1. 包含了开始标签与结束标签，以开始标签起始，以结束标签终止；
  2. 元素的内容是开始标签与结束标签之间的内容；
  3. 某些 HTML 元素具有空内容，空元素在开始标签中进行关闭，如 <br/>；
  4. 大多数 HTML 元素可拥有属性。

HTML 属性：

  1. HTML 元素可以设置属性，属性可以在元素中添加附加信息；
  2. 属性一般描述于开始标签；
  3. 属性总是以名称/值对的形式出现，比如：name="value"；
  4. 属性值应该始终被包括在引号内，双引号是最常用的，不过使用单引号也没有问题。在某些个别的情况下，比如属性值本身就含有双引号，那么您必须使用单引号；
  5. 适用于大多数元素内的属性有：class/id/style/title 等。
  6. 注意：class 属性可多用 class=" "（引号里面可以填入多个class属性）；id 属性只能单独设置 id=" "（只能填写一个，多个无效）

** 举例 ** <!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>html</title>
</head>
<body>
    <p>html主体。</p>
</body>
</html>
解析：
<!DOCTYPE html>：声明为 HTML5 文档；
<html>：HTML 页面的根元素
<head>：包含了文档的元（meta）数据，如 <meta charset="utf-8"> 定义网页编码格式为 utf-8。
<title>：描述了文档的标题
<body>：包含了可见的页面内容

<!DOCTYPE html>：声明为 HTML5 文档

必须位于 HTML5 文档中的第一行

  • HTML 4.01的 strict：<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
  • HTML 4.01 transitional：<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
  • HTML 4.01 Frameset：<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">
  • XHTML 1.0 Strict：<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
  • XHTML 1.0 Transitional：<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
  • XHTML 1.0 Frameset：<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Frameset//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-frameset.dtd">

html： HTML 页面的根元素

限定了文档的开始点和结束点

属性：
lang：支持的语言，如 lang="en"，英语，中文为"zh"。

body：包含了可见的页面内容

定义文档的主体。包含文档的所有内容，比如文本、超链接、图像、表格和列表等等。

head：包含了文档的元（meta）数据

用于定义文档的头部，它是所有头部元素的容器。<head> 中的元素可以引用脚本、指示浏览器在哪里找到样式表、提供元信息等等。文档的头部描述了文档的各种属性和信息，包括文档的标题、在 Web 中的位置以及和其他文档的关系等。绝大多数文档头部包含的数据都不会真正作为内容显示给读者。

可包含 title、style、base、link、meta、script 等元素

title：描述了文档的标题

title 作为元素时描述文档的标题，浏览器工具栏的标题、当网页添加到收藏夹时，显示在收藏夹中的标题、在搜索引擎结果页面的标题，而作为属性时描述了元素的额外信息（作为工具条使用）

** 举例 ** <!DOCTYPE html>
<html>
<head>
  <title>html</title>
</head>
<body>
 <p title="这个就是 title 属性的功能哟">请将鼠标移动到此处</p>
</body>
</html>

a：链接

设置超文本链接。超链接可以是一个字，一个词，或者一组词，也可以是一幅图像。点击这些内容来跳转到新的文档或者当前文档中的某个部分。

把鼠标指针移动到网页中的某个链接上时，箭头会变为一只小手。

默认链接在浏览器中形式为：
1.未访问过的链接显示为蓝色字体并带有下划线；
2.访问过的链接显示为紫色并带有下划线；
3.点击链接时，链接显示为红色并带有下划线。

属性：

  1. href：指定链接地址
    网络链接：href="http://www.baidu.com/"（注意：请始终将正斜杠添加到子文件夹，即若最后不是 .xx 文件则需要在后面加正斜杠）；
    页首位置：href="" 这里是跳转到最前面，页首
    本页指定位置链接：id="demo"   href="#demo" 在文档某个位置设置 id，然后再在另一个地方设置 href，这样点击 href 链接地方之后会跳转到 id 位置
    本地链接：href="X://xx/xx.html" 跳转到另一个页面，这个页面的位置是本地磁盘上的<
  3. target：定义被链接文档在何处显示
    target="_blank"：在新窗口打开；
    target="_top"：直接在本窗口打开，并替换当前整个窗口；
    _parent：在父窗口中打开链接
    _self：默认，当前页面跳转，在相同窗口中打开被链接文档
    framename：在指定的框架中打开被链接文档
  4. id：创建 HTML 文档书签标记，跳转到文档指定位置（若为空则会跳转到页面最顶端），如 id="this1"（此时href="#this1"）可从 href 定义处点击跳转到 id 使用处（也可以直接使用 href="xxx/xx.html#this1" 即从另一个页面跳转到 xx 页面的 this 处）。
  5. rel/rev：rel 属性指定从源文档到目标文档的关系，而 rev 属性则指定从目标文档到源文档的关系。可同时使用
    rel="next" rev="prev"：从源到目标的关系是移动到下一个文档，而从目标到源的关系则是返回前一个文档
    属性值有：
      alternate：文档的可选版本（例如打印页、翻译页或镜像）；
      stylesheet：文档的外部样式表；
      start：集合中的第一个文档；
      next：集合中的下一个文档；
      prev：集合中的前一个文档；
      contents：文档目录；
      index：文档索引；
      glossary：文档中所用字词的术语表或解释；
      copyright：包含版权信息的文档；
      chapter：文档的章；
      section：文档的节；
      subsection：文档的子段；
      appendix：文档附录；
      help：帮助文档；
      bookmark：相关文档；
      nofollow：Google 使用 "nofollow"，用于指定 Google 搜索引擎不要跟踪链接；
      licence：
      tag：
      friend：
  6. type：规定目标 URL 的 MIME 类型，仅在 href 属性存在时使用
    type="text/html"：目标类型为 html
  7. shape/coords：两者配合使用，可规定 <object> 或 <img> 元素中链接的尺寸、形状和位置
    shape="rect"：定义矩形区域
    circle：圆形区域
    poly：多边形区域
    default：全部区域
    coords="x1,y1,x2,y2"：矩形的左上角和右下角坐标
    x,y,radius：圆形的圆心坐标和半径
    x1,y1,x2,y2,...,xn,yn：多边形的各边角坐标
    比如配合 <map>、<object> 创建图像地图
  8. name：指定锚名称；创建（大型）文档内的书签；
    name="C4"：指定锚名称为 C4
  9. charset：指定作为链接目标的文档中所使用的字符编码
    charset="UTF-8"：编码为 utf-8
  10. download：下载链接地址
    download="/xx/xx.png"：文件名称没有限定值，浏览器会自动在文件名称末尾添加该下载文件的后缀（.html .pdf .txt .png 等）
  11. hreflang：指定被链接文档的语言
    hreflang="en"：英语
    注意：和 lang 属性不同的是，hreflang 属性不会指定标签中的内容所使用的语言，而是指定被 href 属性调用的文档所使用的语言
  12. media：规定目标 URL 的媒介类型
    media="print and (resolution:300dpi)"：可打印的媒体属性页
    all：默认值，适合所有设备
    aural：语音合成器
    braille：盲文反馈装置
    handheld：手持设备（小屏幕，有限带宽）
    projection：投影机
    print：打印预览模式/打印页面
    screen：计算机屏幕
    tty：电传打字机以及使用等宽字符网络的类似媒介
    tv：电视类型设备（低分辨率，有限的分页能力）
    width：目标显示区域宽度，可用 min- 和 max- 前缀，如 media="screen and (min-width:500px)"
    height：目标显示区域高度，可用 min- 和 max- 前缀，如 media="screen and (max-height:700px)"
    device-width：目标显示器/纸张宽度，可用 min- 和 max- 前缀，如 media="screen and (device-width:500px)"
    devic-height：目标显示器/纸张高度，可用 min- 和 max- 前缀，如 media="screen and (device-height:500px)"
    orientation：目标显示器/纸张的取向，portrait 或 landscape，如 media="all and (orientation: landscape)"
    aspect-ratio：目标显示区域的宽度/高度比，可用 min- 和 max- 前缀，如 media="screen and (aspect-ratio:16/9)"
    device-aspect-ratio：目标显示器/纸张的 device-width/device-height 比率，可用 min- 和 max- 前缀，如 media="screen and (aspect-ratio:16/9)"
    color：目标显示器的 bits per color，可用 min- 和 max- 前缀，如 media="screen and (color:3)"
    color-index：目标显示器能够处理的颜色数，可用 min- 和 max- 前缀，如 media="screen and (min-color-index:256)"
    monochrome：在单色帧缓冲中的每像素比特，可用 min- 和 max- 前缀，如 media="screen and (monochrome:2)"
    resolution：目标显示器/纸张的像素密度 (dpi or dpcm)，可用 min- 和 max- 前缀，如 media="print and (resolution:300dpi)"
    scan：tv 显示器的扫描方法，"progressive" 和 "interlace"，如 media="tv and (scan:interlace)"
    grid：输出设备是网格还是位图，"1" 代表网格，"0" 是其他，如 media="handheld and (grid:1)"
    可使用的运算符有：and：与；not：非；,（逗号）：或
