Label：标签，用于显示不可编辑的文本或图像

属性
名称 功能
anchor 1.控制文本（或图像）在 Label 中显示的位置；
2. "n", "ne", "e", "se", "s", "sw", "w", "nw",
或者 "center" 来定位（ewsn 代表东西南北，上北下南左西右东）；
3. 默认值是 "center"
activebackground 设置当 Label 处于活动状态（通过 state 选项设置状态）的背景色
activeforeground 设置当 Label 处于活动状态（通过 state 选项设置状态）的前景色
activebitmap 设置当 Label 处于活动状态（通过 state 选项设置状态）的填充的位图
background(bg) 背景色,可以是英文名字，也可以是 RGB 格式
foreground(fg) 前景色
bitmap 1. 指定显示到 Label 上的位图；
2. 如果指定了 image 选项，则该选项被忽略
3. error、info、hourglass、questhead、question、warning、gray12、gray25、gray50、gray75，可直接加载 .xbm 文件，前面需加 @ 符号，如 bitmap="@xx/xxx.xbm"

3. 如果该选项设置为 "center"，文本显示在图像上（文本重叠图像）
4. 如果该选项设置为 "bottom"，"left"，"right" 或 "top"，那么图像显示在文本的旁边（如 "bottom"，则图像在文本的下方）；
5. 默认值是 NONE
cursor 1当鼠标在 Label 上飘过的时候的鼠标样式
hand（握手）、heart（心形）、arrow（箭头），circle（圆形）、clock（钟形）、cross（十字架形）、dotbox（盒子）、exchange（转换形）、fleur（十字箭头形）、man、mouse（鼠标）、plus（+形）、shuttle、sizing、spider、spraycan、star、target、tcross、trek、watch（沙漏）
disabledbackground 当 Label 不可用的时候背景色的颜色
disabledforeground 当 Label 不可用的时候前景色的颜色
disabledbitmap 当 Label 不可用的时候填充的位图
font 字体，如font='Times -10 bold italic underline overstrike'分别表示字体、字号、加粗、斜体、下划线，删除线
highlightbackground 指定当 Label 没有获得焦点的时候高亮边框的颜色
highlightcolor 指定当 Label 获得焦点的时候高亮边框的颜色
highlightthickness 1. 指定高亮边框的宽度；
2. 默认值是 0（不带高亮边框）
image 标签中的图片，该选项优先于 text 和 bitmap 选项
photo=PhotoImage(file="xx.png")
支持：png、gif、jpg 等；不支持：xbm、ico、bmp 等
justify 多行文本的对齐方式，"left"、"right" 或 "center"，默认居中
text 标签中的文本，可以使用'\n'表示换行(若一行放不下必须使用换行符否则后面的文本将看不到)，如果设置了 bitmap 或 image 选项，该选项则被忽略
textvariable 显示文本自动更新，与 StringVar 等配合着用
注意：不可直接 textvariable="xxx" 这样显示不出 xxx 字符信息
padx 1. 指定 Label 水平方向上的额外间距（内容和边框间）；
2. 单位是像素
pady 1. 指定 Label 垂直方向上的额外间距（内容和边框间）；
2. 单位是像素
underline 1. 跟 text 选项一起使用，用于指定哪一个字符添加下划线（例如用于表示键盘快捷键） ；
2. 默认值是 -1；
3. 例如设置为 1，则说明在 Button 的第 2 个字符处画下划线，下标从 0 开始
wraplength 1. 决定 Label 的文本应该被分成多少行；
2. 该选项指定每行的长度，单位是屏幕单元；
3. 默认值是 0
state 指定 Label 的状态，用于控制 Label 如何显示。可选值有：normal（默认）、active、disable
relief flat（默认，无边框，和 Label 显示一样）、sunken（不可以）、raised（往中间凸）、groove（中间凹）、ridge（中间凸）、solid（往中间凹）
takefocus 1. 如果是 True，该 Label 接受输入焦点
2. 默认值是 False

背景由三部分构成：内容区 + 填充区 + 边框。
  内容区参数：width、height 指定区域大小；background 指定背景颜色
  填充区参数：内容区与边框之间间隔大小，单位像素，参数有 padx、pady
  边框参数：样式 relief、边框宽度 borderwidth（单位像素）、
highlightbackground，highlightcolor，highlightthickness 三个边框参数仅在 Label 允许接收焦点的情况下（tackfocus=True），用于设置焦点获取前后高亮边框颜色以及高亮边框宽度

正弦定理：a/sinA=b/sinB=c/sinC
两边一角得到另一边：c*c=a*a+b*b-2*a*b*cosC
'''

exists:指向现有命名字体（如果有）
其他关键字选项（如果指定了 font，则忽略）：
family:字体系列，例如 Courier，Times
size:字体大小
  如果 size 为正数，则解释为以磅为单位的大小。
  如果 size 是负数，则将其绝对值
解释为以像素为单位的大小。
weight：字体强调 (NORMAL, BOLD)（普通，加粗）
slant：ROMAN, ITALIC（正体，斜体）
underline：字体下划线（0 - 无下划线，1 - 有下划线）
overstrike：字体删除线（0 - 无删除线，1 - 有删除线）

ttk：使用时需导入连个库import tkinter和from tkinter import ttk

ttk和tkinter相同点：tkinter有的，ttk也有；tkinter没有的，ttk也有，办法和属性功能还是相同的，此时，若二者都有的组件，ttk将会覆盖Tkinter；ttk有而Tkinter没有的，将采用ttk的特性。。
区别：tkinter 中的fg,bg 在ttk中并不被支持，ttk是通过style这个对象来实现的。

如：
tkinter：
kinter.Label(text="Test", fg="black", bg="white").pack()

ttk：
style = ttk.Style()
style.configure("BW.TLabel", foreground="black", background="white")
ttk.Label(text="Test", style="BW.TLabel").pack()
