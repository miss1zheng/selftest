类型变量

数字：数字数据类型用于存储数值，python支持四种不同的数据类型：int、long、float、complex。，如a=23+3.14j

字符串：由数字、字母和下划线组成。如b="hello world"

列表：可实现集合类的数据结构。可支持数字、字符、字符串甚至列表(嵌套)，以[ ]方括号为标识。如a=[1,2,"3",'45']

元组：类似列表，但元组不能二次赋值，也就是仅只读功能，以( )括号作为标识。如a=(1,2,"3",'45')

字典：字典的每个键值都使用冒号 : 隔开，每个键值对之间使用逗号 , 隔开，以{}花括号作为标识。如a={1:2,"3":'45'}，其中a[1]为2，a["3"]为'45'。
注意：键是唯一的，值可以一样，一旦键设置一样，则后面设置的会覆盖前面设置的。

面向对象

面向对象三要素：封装、继承、多态

类Class：

封装：即在类中的参数，在外部是不可见的，要想获取或设置需要类设置一个外部接口

继承：子类继承了父类/基类中的功能

多态：子类继承父类的同时，若想子类和父类同函数前提下，子类的函数会覆盖父类，这时子类和父类的这个同函数使用时是不同功能实现的

作用域

局部变量：函数内声明定义，与函数外相同名称的其他变量没有任何关系

全局变量 global：当函数内部和外部使用相同的变量时，在函数内部对全局变量进行操作时就可在函数内部对变量进行 global 声明
注意：在子函数里使用全局变量赋值，然后在父函数里进行其他操作时，子函数里需要对这个全局变量进行 global 声明，如下例子：

** global的使用 ** value1 = 0
value = 0
strr=""
def start2():
  global strr
  strr="222"
  print("start2")
def start1():
  #global value #此处可声明也可不声明，貌似没影响
  value1 = 1
  print("start1")
  if value == 1:
    print("value is 1")
  start2()
  print("strr is ",strr) #若start2函数内没有对strr变量进行全局声明，则返回的是空值
if __name__=="__main__":
  #global value #不需要进行 global 声明，否则发生异常报错
  value = 1
  start1()

错误集

  • TypeError: datarecv() argument after * must be an iterable, not socket
    源码：recvid=Thread(target = datarecv, args = (c), name = 'recv')
    解决方式：传入参数需要是一个元组，所以修改一下代码，这里参数后面要加逗号(c,)即可
  • UnboundLocalError: local variable 'data' referenced before assignment
    解决方式：主要是因为变量既可以是全局也可以是局部，没有让解释器清楚变量是全局变量还是局部变量，此时可以在局部变量函数内部加个global xxx对变量进行声明即可。
  • OSError: [WinError 10038] 在一个非套接字上尝试了一个操作。
    源码：c.send(data)
    解决方式：因为socket已关闭，但这里还要将数据发送出去，加上一个try:xxx except Exception as e:xx finally:xxxx即可
  • SyntaxError: name 'file_name' is assigned to before global declaration
    原因：定义一个全局变量，在函数内部使用时加了 global 声明了这是个全局变量
    解决方式：不在内部使用 global 进行声明即可，即函数内去除 global 的声明
