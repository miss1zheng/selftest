简介

CSS 指层叠样式表，使用 CSS 同时控制多重网页的样式和布局。

CSS规则主要组成：选择器以及一条或多条声明，如h1{color:red;font-size:20px;}
选择器通常是需要改变样式的元素。
每条声明由一个属性和一个值组成，属性和值使用冒号隔开。
声明总以分号 ; 结束，而声明总以大括号 {} 括起来。
CSS注释以/*xxx*/来进行注释

样式表

样式表：外部样式表（即通过link标签链接到.css样式）、内部样式表（使用<style></style>建立样式）、内联样式（直接在每个元素的属性设置style）
外部样式表链接如 <link rel="stylesheet" type="text/css" href="my.css"> 链接到my.css
注意：不要在属性值与单位之间留有空格（如："margin-left: 20 px" ），正确的写法是 "margin-left: 20px" 。

若内联、内部、外部样式表三者同时存在，则设置了一样属性的值 内联>内部>外部>浏览器默认样式，而内部没有设置的外部属性则会保留继承下来。如：
外部设置 h3{color:red;text-align:left;font-size:8pt;}
内部设置 h3{text-align:right;font-size:20pt;}
则结果是：h3{color:red;text-align:right;font-size:20pt;}

优先级：内联样式 > id选择器 > 伪类 > 属性选择器 > 类选择器 > 元素（类型）选择器 > 通用选择器（*）
注意：!important规则除外，当这个被应用在一个样式声明中，该样式声明会覆盖CSS中任何其他的声明，无论它处在声明列表中的何处
当一样时后出现的样式表设置要优先于先出现的样式表设置，在同一组属性设置中标有!important规则的优先级最大。如：p{/*两个color属性在同一组*/ color:#00f !important; /* 优先级最大 */ color:#f00; }

css的7种选择器优先级：内联样式 > id 选择器 > 类选择器 = 伪类选择器 = 属性选择器 > 标签选择器 = 伪元素选择器

选择器

id选择器：可以为标有特定id的html元素指定特定样式。
id选择器以“#”来定义，如：使用#xx{x:xxx;xx:xx;}定义，然后在html中元素属性 id="xx" 来进行使用。
注意：id属性不要以数字开头，因为在Mozilla/Firefox浏览器不起作用。

class选择器：用于描述一组元素样式。class可以在多个元素中使用，在html中以class属性表示，
类选择器以点 . 号显示，如.center{text-align:center;}
也可以指定特定的HTML元素使用class，如使所有的p元素使用class="center"让该元素文本居中。p.center {text-align:center;}
类名的第一个字符不能使用数字！它无法在 Mozilla 或 Firefox 中起作用。

分组选择器：在样式表中有很多具有相同样式的元素，此时可使用分组选择器（每个选择器使用逗号 , 隔开），如h1{color:red;}h2{color:red;}p{color:red;}，使用分组选择器之后h1,h2,p{color:red;}

嵌套选择器：适用于选择器内部的选择器的样式
p{}为所有p元素指定一个样式，这里每个p元素都会是此样式
.marked{}为所有class="marked"元素指定一个样式，如<div class="marked">这里的div内部都是是此样式
.marked p{}为所有class="marked"元素内的p元素指定一个样式，如<div class="marked"><p>这里指定的就是div里面的p元素
p.marked{}为所有的class="marked"的p元素指定一个样式，如<p class="marked">这里的p元素就是此样式

组合选择符：说明了两个选择器直接的关系。
四种组合方式：后代选择器（以空格 分隔）、子元素选择器（以大于>分隔）、相邻兄弟选择器（以加号+分隔）、普通兄弟选择器（以波浪~分隔）

后代选择器：用于选取某元素的后代元素，如：
<style>div p{color:red}</style>
<div><p>xxx</p></div>
这里的p就会是style里设置的样式

子元素选择器：相对于后代选择器，只能选择作为某元素子元素的元素，如
<style>div>p{color:yellow;}</style>
<div><p>xx</p></div>
<div><span><p>yy</p></span></div>
这里xx是style样式，而yy不会显示成style样式

相邻兄弟选择器：可选择紧接在另一元素后的元素，且二者有相同父元素。如果需要选择紧接在另一个元素后的元素，而且二者有相同的父元素，可以使用相邻兄弟选择器。如：
<style>div+p{background-color:yellow;}</style>
<div><p>xxx</p></div>
<p>xx</p>
<p>yy</p>
这里只有xx可以显示从style样式，而xxx和yy都不会有style样式

后续兄弟选择器：选取所有指定元素之后的相邻兄弟元素。如：
<style>div~p{color:yellow;}</style>
<p>xxx</p>
<div><p>xx</p></div>
<p>yy</p>
<p>yy</p>
<div><p>xx</p></div>
<p>yy</p>
这里yy都会显示style样式，而xx和xxx不会有style样式

属性选择器：如[title]{color:blue;}，这里是把包含标题 title 的所有元素变蓝

属性和值选择器：如[title=bob]{color:red;}，这里是把包含 titile="bob" 的所有元素变红

属性和值选择器 多值：
1.使用波浪 ~ 分隔属性和值：
[title~=hello] { color:blue; }
<h1 title="hello hai">Hello 嗨</h1>
<p title="student hello">Hello CSS students!</p>
<h2 title="student">Hi CSS students!</h2>
这里h1和p都会变蓝，而h2不会有效
2.使用位或 | 分隔属性和值：
[lang|=en]{color:blue;}
<p lang="en">Hello!</p>
<p lang="en-us">Hi!</p>
<p lang="us">Hi!</p>
这里前面两个都会变蓝，最后一个无效
3.属性选择器无需使用class或id形式：
<style>
  input[type="text"] { width:150px; display:block; margin-bottom:10px; background-color:yellow;}
  input[type="button"] { width:120px; margin-left:35px; display:block;}
</style>
<form method="get">
  username:<input type="text" value="name" size="20">
  password:<input type="text" value="password" size="20">
  <input type="button" value="Button">
</form>

CSS 属性选择器 ~=, |=, ^=, $=, *= 的区别：
"value 是完整单词" 类型的比较符号: ~=, |=
"拼接字符串" 类型的比较符号: *=, ^=, $=
1.attribute 属性中包含 value：
[attribute~=value] 属性中包含独立的单词为 value，如：
[title~=value]{color:red;}
<p title="max value">hello</p>
显示为红色
[attribute*=value] 属性中做字符串拆分，只要能拆出来 value 这个词就行，如：
[title~=value]{color:red;}
<p title="max value">hello</p>
<p title="minvalue">hai</p>
都会显示为红色
2.attribute 属性以 value 开头：
[attribute|=value] 属性中必须是完整且唯一的单词，或者以 - 分隔开，如：
[lang|=en]{color:red;}
<p lang="en-us">hello</p>
这里显示红色
attribute^=value] 属性的前几个字母是 value 就可以，如：
[lang^=en]{color:red;}
<p lang="ennn">hai</p>
这里显示红色
3.attribute 属性以 value 结尾：
[attibute$=value] 属性的后几个字母是 value 就可以，如：
div[class$="test"]{color:red;}
<div class="first_test">第一个 div 元素。</div>
<div class="test">第二个 div 元素。</div>
<div class="test22">第三个 div 元素。</div>
<p class="test">这是段落中的文本。</p>
这里的前两个元素显示红色，后两个元素无效

伪类和伪元素

伪类：用于添加一些选择器的特殊效果。
伪元素：用来添加一些选择器的特殊效果。

伪类的语法：
selector:pseudo-class {property:value;}
CSS类也可以使用伪类：
selector.class:pseudo-class {property:value;}

伪元素的语法：
selector:pseudo-element {property:value;}
CSS类也可以使用伪元素：
selector.class:pseudo-element {property:value;}

伪类和css类配合使用：a.red:visited {color:#FF0000;} <a class="red" href="css-syntax.html">CSS 语法</a>，链接已被访问，它会显示为红色。

anchor伪类：a:link、a:visited、a:hover、a:active

first-child 伪类：选择父元素的第一个子元素。如
p:first-child{color:blue;}
<p>xx</p>
<p>yy</p>
xx有效，yy无效
如：
p > i:first-child{color:blue;}
<<p>x<i>ii</i>xx</p>
<p>x<i>ii</i>xx<i>iiii</i>x</p>
这里只有ii显示出style样式
如：
p:first-child i{color:blue;}
<p>x<i>ii</i>xx<i>ii</i>x</p>
<p>x<i>iii</i>xx<i>iii</i>x</p>
只有ii有效，即选择器匹配所有作为元素的第一个子元素的 <p> 元素中的所有 <i> 元素

:lang 伪类：为不同的语言定义特殊的规则。如：
q:lang(no){quotes: "~" "~";}
<p>Some text <q lang="no">A quote in a paragraph</q> Some text.</p>
在q的开始和结束位置上加入了 ~ 符号

:first-line 伪元素：用于向文本的首行设置特殊样式。如：
p:first-line {color:red;}
<p>xx</p>
xx显示为红色
注意："first-line" 伪元素只能用于块级元素。

:first-letter 伪元素：用于向文本的首字母设置特殊样式，如：
p:first-letter {color:red;}
<p>xxx</p>
首个x会变红色
注意："first-letter" 伪元素只能用于块级元素。

伪元素和css类配合使用：如：
p.article:first-letter {color:#ff0000;}
<p class="article">文章段落</p>
首字母变为红色

多个伪元素：如：
p:first-letter{color:red}
p:first-line {color:blue;}
<p>你可以结合使用"first-line"和"first-letter"伪元素向文本的首行和首字母设置特殊样式。</p>
第一个字为红色，首行是蓝色

:before 伪元素：可以在元素的内容前面插入新内容。如：
h1:before {content:url(smiley.gif);}
<h1>xx</h1>
在xx文本前面有个.gif图像

:after 伪元素：可以在元素的内容之后插入新内容。如：
h1:after {content:url(smiley.gif);}
<h1>xx</h1>
在xx文本后面有个.gif图像

链接

四个链接状态：
a:link：正常，未访问过的链接
a:visited：用户已访问过的链接
a:hover：用户鼠标发在链接时
a:active：链接被点击的那刻
注意：当多个样式设置时，hover必须在link和visited后面，而active必须在hover后面

列表

    2D转换

    可以对元素进行移动、缩放、转动、拉长或拉伸。使用的是transform元素
    转换的效果是让某个元素改变形状，大小和位置。

    方法：translate()、rotate()、scale()、skew()、matrix()

    translate()：根据左(X轴)和顶部(Y轴)位置给定的参数，从当前元素位置移动。如transform: translate(50px,100px);
    rotate()：在一个给定度数顺时针旋转的元素。负值是允许的，这样是元素逆时针旋转。如transform: rotate(30deg);/*顺时针旋转30度。*/
    scale()：该元素增加或减少的大小，取决于宽度（X轴）和高度（Y轴）的参数，如transform: scale(2,3);/*宽两倍高三倍*/
    skew()：语法transform:skew( [,]);，分别表示X轴和Y轴倾斜的角度，如果第二个参数为空，则默认为0，参数为负表示向相反方向倾斜。
    matrix()：有六个参数，包含旋转，缩放，移动（平移）和倾斜功能。如matrix(0.866,0.5,-0.5,0.866,0,0);

    动画

    要创建 CSS3 动画，你需要了解 @keyframes 规则。
    @keyframes 规则是创建动画。指定一个 CSS 样式和动画将逐步从目前的样式更改为新的样式。比如：
    @keyframes myfirst{from {background: red;}to {background: yellow;}}
    @-webkit-keyframes myfirst /* Safari 与 Chrome */
    {from {background: red;}to {background: yellow;}}
    div{animation: myfirst 5s;-webkit-animation: myfirst 5s; /* Safari 与 Chrome */}

    当在 @keyframes 创建动画，把它绑定到一个选择器，否则动画不会有任何效果。
    至少这两个CSS3的动画属性绑定向一个选择器：规定动画的名称、规定动画的时长。如animation: myfirst 5s;

    动画是使元素从一种样式逐渐变化为另一种样式的效果。
    可以改变任意多的样式任意多的次数。
    可用百分比来规定变化发生的时间，或用关键词 "from" 和 "to"，等同于 0% 和 100%。
    0% 是动画的开始，100% 是动画的完成。
    如：@keyframes myfirst{
    0% {background: red; left:0px; top:0px;}
    25% {background: yellow; left:200px; top:0px;}
    50% {background: blue; left:200px; top:200px;}
    75% {background: green; left:0px; top:200px;}
    100% {background: red; left:0px; top:0px;}}

    属性描述
    @keyframes规定动画
    animation所有动画属性的简写属性
    animation-name规定 @keyframes 动画的名称
    animation-duration规定动画完成一个周期所花费的秒或毫秒。默认是 0
    animation-timing-function规定动画的速度曲线。默认是 "ease"
    animation-fill-mode规定当动画不播放时（当动画完成时，或当动画有一个延迟未开始播放时），要应用到元素的样式
    animation-delay规定动画何时开始。默认是 0
    animation-iteration-count规定动画被播放的次数。默认是 1
    animation-direction规定动画是否在下一周期逆向地播放。默认是 "normal"
    animation-play-state规定动画是否正在运行或暂停。默认是 "running"

    多媒体查询

    @media 规则：viewport(视窗) 的宽度与高度、设备的宽度与高度、朝向 (智能手机横屏，竖屏) 、分辨率

    语法：@media not|only mediatype and (expressions) {CSS 代码...;}，多媒体查询由多种媒体组成，可以包含一个或多个表达式，表达式根据条件是否成立返回 true 或 false。

    如果指定的多媒体类型匹配设备类型则查询结果返回 true，文档会在匹配的设备上显示指定样式效果。
    除非你使用了 not 或 only 操作符，否则所有的样式会适应在所有设备上显示效果。
    not: not是用来排除掉某些特定的设备的，比如 @media not print（非打印设备）。
    only: 用来定某种特别的媒体类型。对于支持Media Queries的移动设备来说，如果存在only关键字，移动设备的Web浏览器会忽略only关键字并直接根据后面的表达式应用样式文件。对于不支持Media Queries的设备但能够读取Media Type类型的Web浏览器，遇到only关键字时会忽略这个样式文件。
    all: 所有设备，这个应该经常看到。
    如：link rel="stylesheet" media="mediatype and|not|only (expressions)" href="print.css">

    类型：all：用于所有多媒体类型设备
    print：用于打印机
    screen：用于电脑屏幕，平板，智能手机等
    speech：用于屏幕阅读器

    分页

    如果你的网站有很多个页面，你就需要使用分页来为每个页面做导航。

    align-content：修改 flex-wrap 属性的行为

    类似于 align-items, 但它不是设置弹性子元素的对齐，而是设置各个行的对齐。

    stretch：默认。各行将会伸展以占用剩余的空间。
    flex-start：各行向弹性盒容器的起始位置堆叠。
    flex-end：各行向弹性盒容器的结束位置堆叠。
    center：各行向弹性盒容器的中间位置堆叠。
    space-between：各行在弹性盒容器中平均分布。

    Gradients：渐变

    线性渐变和径向渐变
    线性渐变：向下/向上/向左/向右/对角方向
    径向渐变：由中心定义。
    线性渐变相关属性：background-image

    线性渐变，你必须至少定义两种颜色节点。颜色节点即你想要呈现平稳过渡的颜色。同时，你也可以设置一个起点和一个方向（或一个角度）。

    线性语法：background-image: linear-gradient(direction, color-stop1, color-stop2, ...);
    从上到下：background-image: linear-gradient(red, blue);
    从左到右：height: 200px;background-image: linear-gradient(to right, red , yellow);
    对角：过指定水平和垂直的起始位置来制作一个对角渐变。height: 200px;background-image: linear-gradient(to bottom right, red, yellow);左上到右下

    加入角度语法：background-image: linear-gradient(angle, color-stop1, color-stop2);
    角度是指水平线和渐变线之间的角度，逆时针方向计算。换句话说，0deg 将创建一个从下到上的渐变，90deg 将创建一个从左到右的渐变。
    background-image: linear-gradient(-90deg, red, yellow);

    多颜色：background-image: linear-gradient(red, yellow, green);
    background-image: linear-gradient(to right, red,orange,yellow,green,blue,indigo,violet);带有彩虹颜色和文本的线性渐变

    透明度：background-image: linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1));从左边开始的线性渐变。起点是完全透明，慢慢过渡到完全不透明的红色
    为了添加透明度，我们使用 rgba() 函数来定义颜色节点。rgba() 函数中的最后一个参数可以是从 0 到 1 的值，它定义了颜色的透明度：0 表示完全透明，1 表示完全不透明。

    重复的线性渐变：background-image: repeating-linear-gradient(red, yellow 10%, green 20%);

    径向渐变：径向渐变由它的中心定义。为了创建一个径向渐变，你也必须至少定义两种颜色节点。颜色节点即你想要呈现平稳过渡的颜色。同时，你也可以指定渐变的中心、形状（圆形或椭圆形）、大小。默认情况下，渐变的中心是 center（表示在中心点），渐变的形状是 ellipse（表示椭圆形），渐变的大小是 farthest-corner（表示到最远的角落）。

    径向语法：background-image: radial-gradient(shape size at position, start-color, ..., last-color);

    颜色节点均匀分布的径向渐变：background-image: radial-gradient(red, yellow, green);

    颜色节点不均匀分布的径向渐变：background-image: radial-gradient(red 5%, yellow 15%, green 60%);

    设置形状：shape 参数定义了形状。它可以是值 circle 或 ellipse。其中，circle 表示圆形，ellipse 表示椭圆形。默认值是 ellipse。如 background-image: radial-gradient(circle, red, yellow, green);

    不同尺寸大小关键字的使用：closest-side、farthest-side、closest-corner、farthest-corner
    带有不同尺寸大小关键字的径向渐变
    background-image: radial-gradient(closest-side at 60% 55%, red, yellow, black);
    background-image: radial-gradient(farthest-side at 60% 55%, red, yellow, black);

    重复的径向渐变：background-image: repeating-radial-gradient(red, yellow 10%, green 15%);

    transition：一种样式逐渐改变为另一种的效果

    要实现这一点，必须规定两项内容：指定要添加效果的CSS属性；指定效果的持续时间。如transition: width 2s;
    注意： 如果未指定的期限，transition将没有任何效果，因为默认值是0。

    多项改变：要添加多个样式的变换效果，添加的属性由逗号分隔，如transition: width 2s, height 2s, transform 2s;

    属性描述
    transition简写属性，用于在一个属性中设置四个过渡属性
    transition-property规定应用过渡的 CSS 属性的名称
    transition-duration定义过渡效果花费的时间。默认是 0
    transition-timing-function规定过渡效果的时间曲线。默认是 "ease"
    transition-delay规定过渡效果何时开始。默认是 0

    transition-delay：指定何时将开始切换效果。

    time：指定秒或毫秒数之前要等待切换效果开始

    transition-property

    transition-duration：动画耗费时间

    time：规定完成过渡效果需要花费的时间（以秒或毫秒计）。 默认值是 0，意味着不会有效果。

    transition-property：指定CSS属性的nametransition效果（transition效果时将会启动指定的CSS属性的变化）。

    提示:一个转场效果，通常会出现在当用户将鼠标悬停在一个元素上时。
    注意：始终指定transition-duration属性，否则持续时间为0，transition不会有任何效果。

    none：没有属性会获得过渡效果
    all：所有属性都将获得过渡效果
    property：定义应用过渡效果的 CSS 属性名称列表，列表以逗号分隔。

    transition-timing-function：指定切换效果的速度。

    此属性允许一个过渡效果，以改变其持续时间的速度。

    linear：规定以相同速度开始至结束的过渡效果（等于 cubic-bezier(0,0,1,1)）
    ease：规定慢速开始，然后变快，然后慢速结束的过渡效果（cubic-bezier(0.25,0.1,0.25,1)）
    ease-in：规定以慢速开始的过渡效果（等于 cubic-bezier(0.42,0,1,1)）
    ease-out：规定以慢速结束的过渡效果（等于 cubic-bezier(0,0,0.58,1)）
    ease-in-out：规定以慢速开始和结束的过渡效果（等于 cubic-bezier(0.42,0,0.58,1)）
    cubic-bezier(n,n,n,n)：在 cubic-bezier 函数中定义自己的值。可能的值是 0 至 1 之间的数值。

    break-all：允许在单词内换行

    @keyframes：创建动画

    创建动画是通过逐步改变从一个CSS样式设定到另一个。
    在动画过程中，您可以更改CSS样式的设定多次。
    指定的变化时发生时使用％，或关键字"from"和"to"，这是和0％到100％相同。
    0％是开头动画，100％是当动画完成。
    为了获得最佳的浏览器支持，您应该始终定义为0％和100％的选择器。
    注意: 使用animation属性来控制动画的外观，还使用选择器绑定动画。

    语法：@keyframes animationname {keyframes-selector {css-styles;}}

    animationname：必需的。定义animation的名称
    keyframes-selector：必需的。动画持续时间的百分比
    合法值：0-100%、from (和0%相同)、to (和100%相同)
    注意： 您可以用一个动画keyframes-selectors。
    css-styles：必需的。一个或多个合法的CSS样式属性

    @media：自定义媒体样式

    @media 可以针对不同的屏幕尺寸设置不同的样式，特别是如果你需要设置设计响应式的页面，@media 是非常有用的。当你重置浏览器大小的过程中，页面也会根据浏览器的宽度和高度重新渲染页面。

    语法：@media mediatype and|not|only (media feature) {CSS-Code;}
    <link rel="stylesheet" media="mediatype and|not|only (media feature)" href="mystylesheet.css">
