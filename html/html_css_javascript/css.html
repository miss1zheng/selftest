简介

CSS 指层叠样式表，使用 CSS 同时控制多重网页的样式和布局。

CSS规则主要组成：选择器以及一条或多条声明，如h1{color:red;font-size:20px;}
选择器通常是需要改变样式的元素。
每条声明由一个属性和一个值组成，属性和值使用冒号隔开。
声明总以分号 ; 结束，而声明总以大括号 {} 括起来。
CSS注释以/*xxx*/来进行注释

id选择器：可以为标有特定id的html元素指定特定样式。
id选择器以“#”来定义，如：使用#xx{x:xxx;xx:xx;}定义，然后在html中元素属性 id="xx" 来进行使用。
注意：id属性不要以数字开头，因为在Mozilla/Firefox浏览器不起作用。

class选择器：用于描述一组元素样式。class可以在多个元素中使用，在html中以class属性表示，
类选择器以点 . 号显示，如.center{text-align:center;}
也可以指定特定的HTML元素使用class，如使所有的p元素使用class="center"让该元素文本居中。p.center {text-align:center;}
类名的第一个字符不能使用数字！它无法在 Mozilla 或 Firefox 中起作用。

样式表：外部样式表（即通过link标签链接到.css样式）、内部样式表（使用<style></style>建立样式）、内联样式（直接在每个元素的属性设置style）
外部样式表链接如 <link rel="stylesheet" type="text/css" href="my.css"> 链接到my.css
注意：不要在属性值与单位之间留有空格（如："margin-left: 20 px" ），正确的写法是 "margin-left: 20px" 。

若内联、内部、外部样式表三者同时存在，则设置了一样属性的值 内联>内部>外部>浏览器默认样式，而内部没有设置的外部属性则会保留继承下来。如：
外部设置 h3{color:red;text-align:left;font-size:8pt;}
内部设置 h3{text-align:right;font-size:20pt;}
则结果是：h3{color:red;text-align:right;font-size:20pt;}

优先级：内联样式 > id选择器 > 伪类 > 属性选择器 > 类选择器 > 元素（类型）选择器 > 通用选择器（*）
注意：!important规则除外，当这个被应用在一个样式声明中，该样式声明会覆盖CSS中任何其他的声明，无论它处在声明列表中的何处
当一样时后出现的样式表设置要优先于先出现的样式表设置，在同一组属性设置中标有!important规则的优先级最大。如：p{/*两个color属性在同一组*/ color:#00f !important; /* 优先级最大 */ color:#f00; }

css的7种选择器优先级：内联样式 > id 选择器 > 类选择器 = 伪类选择器 = 属性选择器 > 标签选择器 = 伪元素选择器

链接

四个链接状态：
a:link：正常，未访问过的链接
a:visited：用户已访问过的链接
a:hover：用户鼠标发在链接时
a:active：链接被点击的那刻
注意：当多个样式设置时，hover必须在link和visited后面，而active必须在hover后面

列表

在css中定义后，在html使用为class="xx"，如css中为ul.a{url('other/apple.png');}或ol.b{list-style-type: circle;}，则在html中引用为<ul class="a">

在所有浏览器中都能显示图像：css中为
ul{
  list-style-type: none;

  margin: 0px;}
ul li{

<ul><li>hello</li></ul>

默认情况下列表 <ul> 或 <ol> 还设置了内边距和外边距，可使用 margin:0 和 padding:0 来移除

background：背景设置

优先级为 color > image > position/size > repeat > origin > clip > attachment > 默认/继承。整个背景简写如background:#ffffff url('img_tree.png') no-repeat fixed right top;

background-attachment：背景是否滚动

scroll：背景图随页面的滚动而滚动，默认
fixed：背景图不随页面滚动而滚动
local：背景图随元素内容的滚动二滚动
initial：默认值
inherit：从父元素继承

background-clip：背景图的绘画区域

border-box：背景绘制在边框方框内（剪切成边框方框），默认值
padding-box：背景绘制在衬距方框内（剪切成衬距方框）
content-box：背景绘制在内容方框内（剪切成内容方框）

background-color：背景色

1.名称：red、green、yellow、gray
2.RGB值：rgb(255,0,0)
3.十六进制：#ffffff
transparent：背景色透明，默认值
inherit：从父元素继承

此背景色还可以用于标识文字的背景色，相当于标亮

background-image：背景图

背景图片设置：url('xx.png')
none：无背景图，默认
linear-gradient()：创建一个线性渐变图（从上到下）
radial-gradient()：用径向渐变创建图像（中间到边缘）
repeating-linear-gradient()：创建重复的线性渐变图
repeating-radial-gradient()：创建重复的径向渐变图
inherit：从父元素继承

多层背景图层叠：
#example1 {
background-image: url(img_flwr.gif), url(paper.gif);/*第一个在第二个的上面*/
background-position: right bottom, left top;
background-repeat: no-repeat, repeat;
padding: 15px;
}

background-origin：背景图的定位区域

padding-box：背景图填充框的相对位置
border-box：背景图边界框的相对位置
content-box：背景图的相对位置的内容框

background-position：背景图片起始位置

left top
left center
left bottom
right top
right center
right bottom
center top
center center
center bottom
x% x%：第一个为水平位置，第二个十垂直位置。默认0% 0%（左上角，100% 100%为右下角，若指定一个值则其他值为50%）
注意：若只指定一个关键字，其他值将会是center
xpos ypos：水平位置和垂直位置，左上角为0，单位可以是像素px、em、ex、cm、mm、in、pt、pc、ch、rem、wh、vw、vmin、vmax等
inherit：从父元素继承

background-repeat：水平或垂直平铺（重复）

repeat：在页面水平或垂直方向平铺，默认
repeat-x：只水平平铺
repeat-y：只垂直平铺
no-repeat：不平铺
inherit：从父元素继承

background-size：背景图大小

length：背景图高宽，第一个值为宽度，第二个值为高度，若只给一个值则第二个值默认auto（自动）
percentage：百分比，第一个宽，第二个高，若只设置一个，则第二个默认auto，如100% 100%
cover：保持图像的纵横比并将图像缩放成将完全覆盖背景定位区域的最小大小
contain：保持图像的纵横比并将图像缩放成将适合背景定位区域的最大大小

color：文本颜色

名称：red、green、yellow、gray、blue、black、white、lightseagreen
十六进制：#0000ff（6bit或3bit）
RGB值：rgb(255,255,255)、rgb(100%,0%,30%)
RGBA（红绿蓝和阿尔法）：rgba(255,0,0,0.2)，其中0.2表示20%不透明，若为1则不透明，为0则透明
HSL：色相（色环，一个彩色圆环）、饱和度（百分比，100%满饱和，0%灰度）、明度（百分比，100%白色，0%黑色），如hsl(120,100%,30%)
HSLA：比HSL多了一个阿尔法（0透明，1不透明），如hsla(120,100%,30%,0.05)

一个网页的背景色是指在主体内的选择，如body{color:red;}

text-align：文本对齐方式

center、left、right、justify（两端对齐）、inherit

text-decoration：向文本添加修饰

添加到文本的修饰，下划线、上划线、删除线等。包括三种属性：text-decoration-line、text-decoration-color、text-decoration-style

none：没有文本装饰，删除链接下划线，默认
overline dotted：文本点点上划线
line-through：文本删除线
underline wavy red：红色波浪下划线
blink：闪烁文本
inherit：从父元素继承
unset：
initial：

text-decoration-color：文本修饰颜色

规定文本修饰（下划线 underline、上划线 overline、中划线 line-through）的颜色。
注意：只有在带有可见的 text-decoration 的元素上，text-decoration-color 属性才起作用。

color：red、green等
initial：默认值
inherit：从父元素继承

text-decoration-line：文本修饰划线

none：默认值。规定文本修饰没有线条
underline：文本下划线
overline：文本上划线
line-through：规定文本的中间将显示一条线
initial：设置该属性为它的默认值
inherit：从父元素继承该属性

text-decoration-style：文本修饰风格

solid：默认值。线条将显示为单线
double：线条将显示为双线
dotted：线条将显示为点状线
dashed：线条将显示为虚线
wavy：线条将显示为波浪线
initial：设置该属性为它的默认值
inherit：从父元素继承该属性

text-transform：文本转换

用来指定在一个文本中的大写和小写字母，或每个单词的首字母大写。

none：默认，带有大小写字母的标准文本
uppercase：都是大写
lowercase：都是小写
capitalize：单词首字母大写
inherit：从父元素继承

text-indent：文本首行缩进

用来指定文本的第一行的缩进。
注意：负值是允许的。如果值是负数，将第一行左缩进。

length：固定缩进
百分比：基于父元素宽度的百分比缩进
inherit：从父元素继承

text-shadow：文字阴影

优先级：h-shadow > v-shadow > blur > colo

h-shadow：必需。水平阴影的位置。允许负值。
v-shadow：必需。垂直阴影的位置。允许负值。
blur：可选。模糊的距离。
color：可选。阴影的颜色

inherit：从父元素继承

inherit：从父元素继承

word-spacing：字符间距

normal：默认
length：字符固定空间（允许使用负值），如30px，0.3，2等
inherit：从父元素继承

white-space：空白处理

normal：默认。空白会被浏览器忽略。
pre：空白会被浏览器保留。其行为方式类似 HTML 中的 <pre> 标签
nowrap：文本不会换行，文本会在在同一行上继续，直到遇到
标签为止

pre-line：合并空白符序列，但是保留换行符
inherit 规定应该从父元素继承

vertical-align：设置元素的垂直对齐

该属性定义行内元素的基线相对于该元素所在行的基线的垂直对齐。允许指定负长度值和百分比值。这会使元素降低而不是升高。在表单元格中，这个属性会设置单元格框中的单元格内容的对齐方式。

baseline：默认。元素放置在父元素的基线上。
sub：垂直对齐文本的下标。
super：垂直对齐文本的上标
top：把元素的顶端与行中最高元素的顶端对齐
text-top：把元素的顶端与父元素字体的顶端对齐
middle：把此元素放置在父元素的中部。
bottom：使元素及其后代元素的底部与整行的底部对齐。
text-bottom：把元素的底端与父元素字体的底端对齐。
length：将元素升高或降低指定的高度，可以是负数。
百分比：使用 "line-height" 属性的百分比值来排列此元素。允许使用负值。
inherit：从父元素继承

font：字体设置

优先级：font-style > font-variant > font-weight > font-size/line-height > font-family
font-size和font-family的值是必需的。如果缺少了其他值，默认值将被插入，如果有默认值的话。如 15px arial,sans-serif 或 font:italic bold 12px/30px Georgia,serif

font-family：字体设置

如果字体系列的名称超过一个字，它必须用引号，如"宋体"、"Times New Roman"等。
多个字体系列使用逗号 , 分隔，如 font-family:"宋体",Times,serif

宋体、楷体、黑体、verdana

outline：轮廓

绘制于元素周围的一条线，位于边框边缘的外围，可起到突出元素的作用。
顺序：outline-color > outline-style > outline-width

outline-color：轮廓颜色
outline-style：轮廓样式
outline-width：轮廓宽度
inherit：从父元素继承

outline-style：轮廓样式

dotted：点线轮廓
dashed：虚线轮廓
solid：实线轮廓
double：双线轮廓
groove：凹槽轮廓
ridge：垄状轮廓
inset：嵌入轮廓
outset：外凸轮廓

outline-color：轮廓颜色

名称，如red
RGB：如rgb(255,0,0)
十六进制：如#ff0000
invert：颜色反转（逆向的颜色），可使轮廓在不同的背景颜色中都是可见，默认值
inherit：从父元素继承

outline-width：轮廓宽度

绘制于元素周围的一条线，位于边框边缘的外围。
注意：请始终在outline-width属性之前声明outline-style属性。

thin：细轮廓。
medium：默认。中等轮廓。
thick：粗轮廓。
length：自定义设置
inherit：从父元素继承
