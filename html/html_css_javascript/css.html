简介

CSS 指层叠样式表，使用 CSS 同时控制多重网页的样式和布局。

CSS规则主要组成：选择器以及一条或多条声明，如h1{color:red;font-size:20px;}
选择器通常是需要改变样式的元素。
每条声明由一个属性和一个值组成，属性和值使用冒号隔开。
声明总以分号 ; 结束，而声明总以大括号 {} 括起来。
CSS注释以/*xxx*/来进行注释

样式表

样式表：外部样式表（即通过link标签链接到.css样式）、内部样式表（使用<style></style>建立样式）、内联样式（直接在每个元素的属性设置style）
外部样式表链接如 <link rel="stylesheet" type="text/css" href="my.css"> 链接到my.css
注意：不要在属性值与单位之间留有空格（如："margin-left: 20 px" ），正确的写法是 "margin-left: 20px" 。

若内联、内部、外部样式表三者同时存在，则设置了一样属性的值 内联>内部>外部>浏览器默认样式，而内部没有设置的外部属性则会保留继承下来。如：
外部设置 h3{color:red;text-align:left;font-size:8pt;}
内部设置 h3{text-align:right;font-size:20pt;}
则结果是：h3{color:red;text-align:right;font-size:20pt;}

优先级：内联样式 > id选择器 > 伪类 > 属性选择器 > 类选择器 > 元素（类型）选择器 > 通用选择器（*）
注意：!important规则除外，当这个被应用在一个样式声明中，该样式声明会覆盖CSS中任何其他的声明，无论它处在声明列表中的何处
当一样时后出现的样式表设置要优先于先出现的样式表设置，在同一组属性设置中标有!important规则的优先级最大。如：p{/*两个color属性在同一组*/ color:#00f !important; /* 优先级最大 */ color:#f00; }

css的7种选择器优先级：内联样式 > id 选择器 > 类选择器 = 伪类选择器 = 属性选择器 > 标签选择器 = 伪元素选择器

选择器

id选择器：可以为标有特定id的html元素指定特定样式。
id选择器以“#”来定义，如：使用#xx{x:xxx;xx:xx;}定义，然后在html中元素属性 id="xx" 来进行使用。
注意：id属性不要以数字开头，因为在Mozilla/Firefox浏览器不起作用。

class选择器：用于描述一组元素样式。class可以在多个元素中使用，在html中以class属性表示，
类选择器以点 . 号显示，如.center{text-align:center;}
也可以指定特定的HTML元素使用class，如使所有的p元素使用class="center"让该元素文本居中。p.center {text-align:center;}
类名的第一个字符不能使用数字！它无法在 Mozilla 或 Firefox 中起作用。

分组选择器：在样式表中有很多具有相同样式的元素，此时可使用分组选择器（每个选择器使用逗号 , 隔开），如h1{color:red;}h2{color:red;}p{color:red;}，使用分组选择器之后h1,h2,p{color:red;}

嵌套选择器：适用于选择器内部的选择器的样式
p{}为所有p元素指定一个样式，这里每个p元素都会是此样式
.marked{}为所有class="marked"元素指定一个样式，如<div class="marked">这里的div内部都是是此样式
.marked p{}为所有class="marked"元素内的p元素指定一个样式，如<div class="marked"><p>这里指定的就是div里面的p元素
p.marked{}为所有的class="marked"的p元素指定一个样式，如<p class="marked">这里的p元素就是此样式

组合选择符：说明了两个选择器直接的关系。
四种组合方式：后代选择器（以空格 分隔）、子元素选择器（以大于>分隔）、相邻兄弟选择器（以加号+分隔）、普通兄弟选择器（以波浪~分隔）

后代选择器：用于选取某元素的后代元素，如：
<style>div p{color:red}</style>
<div><p>xxx</p></div>
这里的p就会是style里设置的样式

子元素选择器：相对于后代选择器，只能选择作为某元素子元素的元素，如
<style>div>p{color:yellow;}</style>
<div><p>xx</p></div>
<div><span><p>yy</p></span></div>
这里xx是style样式，而yy不会显示成style样式

相邻兄弟选择器：可选择紧接在另一元素后的元素，且二者有相同父元素。如果需要选择紧接在另一个元素后的元素，而且二者有相同的父元素，可以使用相邻兄弟选择器。如：
<style>div+p{background-color:yellow;}</style>
<div><p>xxx</p></div>
<p>xx</p>
<p>yy</p>
这里只有xx可以显示从style样式，而xxx和yy都不会有style样式

后续兄弟选择器：选取所有指定元素之后的相邻兄弟元素。如：
<style>div~p{color:yellow;}</style>
<p>xxx</p>
<div><p>xx</p></div>
<p>yy</p>
<p>yy</p>
<div><p>xx</p></div>
<p>yy</p>
这里yy都会显示style样式，而xx和xxx不会有style样式

伪类和伪元素

伪类：用于添加一些选择器的特殊效果。
伪元素：用来添加一些选择器的特殊效果。

伪类的语法：
selector:pseudo-class {property:value;}
CSS类也可以使用伪类：
selector.class:pseudo-class {property:value;}

伪元素的语法：
selector:pseudo-element {property:value;}
CSS类也可以使用伪元素：
selector.class:pseudo-element {property:value;}

伪类和css类配合使用：a.red:visited {color:#FF0000;} <a class="red" href="css-syntax.html">CSS 语法</a>，链接已被访问，它会显示为红色。

anchor伪类：a:link、a:visited、a:hover、a:active

first-child 伪类：选择父元素的第一个子元素。如
p:first-child{color:blue;}
<p>xx</p>
<p>yy</p>
xx有效，yy无效
如：
p > i:first-child{color:blue;}
<<p>x<i>ii</i>xx</p>
<p>x<i>ii</i>xx<i>iiii</i>x</p>
这里只有ii显示出style样式
如：
p:first-child i{color:blue;}
<p>x<i>ii</i>xx<i>ii</i>x</p>
<p>x<i>iii</i>xx<i>iii</i>x</p>
只有ii有效，即选择器匹配所有作为元素的第一个子元素的 <p> 元素中的所有 <i> 元素

:lang 伪类：为不同的语言定义特殊的规则。如：
q:lang(no){quotes: "~" "~";}
<p>Some text <q lang="no">A quote in a paragraph</q> Some text.</p>
在q的开始和结束位置上加入了 ~ 符号

:first-line 伪元素：用于向文本的首行设置特殊样式。如：
p:first-line {color:red;}
<p>xx</p>
xx显示为红色
注意："first-line" 伪元素只能用于块级元素。

:first-letter 伪元素：用于向文本的首字母设置特殊样式，如：
p:first-letter {color:red;}
<p>xxx</p>
首个x会变红色
注意："first-letter" 伪元素只能用于块级元素。

伪元素和css类配合使用：如：
p.article:first-letter {color:#ff0000;}
<p class="article">文章段落</p>
首字母变为红色

多个伪元素：如：
p:first-letter{color:red}
p:first-line {color:blue;}
<p>你可以结合使用"first-line"和"first-letter"伪元素向文本的首行和首字母设置特殊样式。</p>
第一个字为红色，首行是蓝色

:before 伪元素：可以在元素的内容前面插入新内容。如：
h1:before {content:url(smiley.gif);}
<h1>xx</h1>
在xx文本前面有个.gif图像

:after 伪元素：可以在元素的内容之后插入新内容。如：
h1:after {content:url(smiley.gif);}
<h1>xx</h1>
在xx文本后面有个.gif图像

链接

四个链接状态：
a:link：正常，未访问过的链接
a:visited：用户已访问过的链接
a:hover：用户鼠标发在链接时
a:active：链接被点击的那刻
注意：当多个样式设置时，hover必须在link和visited后面，而active必须在hover后面

列表

在css中定义后，在html使用为class="xx"，如css中为ul.a{url('other/apple.png');}或ol.b{list-style-type: circle;}，则在html中引用为<ul class="a">

在所有浏览器中都能显示图像：css中为
ul{
  list-style-type: none;
  padding: 0px;
  margin: 0px;}
ul li{
  background-image: url('other/apple.png');
  background-repeat: no-repeat;
  background-position: 0px 5px;
  padding-left: 14px; }
在html中为
<ul><li>hello</li></ul>

默认情况下列表 <ul> 或 <ol> 还设置了内边距和外边距，可使用 margin:0 和 padding:0 来移除

background：背景设置

优先级为 color > image > position/size > repeat > origin > clip > attachment > 默认/继承。整个背景简写如background:#ffffff url('img_tree.png') no-repeat fixed right top;

background-attachment：背景是否滚动

scroll：背景图随页面的滚动而滚动，默认
fixed：背景图不随页面滚动而滚动

visibility与display的隐藏元素区别：
visibility:hidden可以隐藏某个元素，但隐藏的元素仍需占用与未隐藏之前一样的空间。也就是说，该元素虽然被隐藏了，但仍然会影响布局。
display:none可以隐藏某个元素，且隐藏的元素不会占用任何空间。也就是说，该元素不但被隐藏了，而且该元素原本占用的空间也会从页面布局中消失。

    position：定位

    static：默认值，即没有定位，遵循正常的文档流对象，不会受到 top, bottom, left, right或z-index影响。
    relative：相对定位元素的定位是相对其正常位置
    fixed：元素的位置相对于浏览器窗口是固定位置，即使窗口是滚动的它也不会移动，移动相对定位元素，但它原本所占的空间不会改变，相对定位元素经常被用来作为绝对定位元素的容器块。元素的位置通过 "left", "top", "right" 以及 "bottom" 属性进行规定
    absolute：绝对定位的元素的位置相对于最近的已定位父元素，如果元素没有已定位的父元素，那么它的位置相对于<html>。元素的位置通过 "left", "top", "right" 以及 "bottom" 属性进行规定
    sticky：粘性定位，基于用户的滚动位置来定位，依赖于用户的滚动，在 position:relative 与 position:fixed 定位之间切换。它的行为就像 position:relative; 而当页面滚动超出目标区域时，它的表现就像 position:fixed;，它会固定在目标位置。元素定位表现为在跨越特定阈值前为相对定位，之后为固定定位。这个特定阈值指的是 top, right, bottom 或 left 之一，换言之，指定 top, right, bottom 或 left 四个阈值其中之一，才可使粘性定位生效。否则其行为与相对定位相同。
    initial：设置该属性为它的默认值
    inherit：从父元素继承

    z-index：元素堆叠顺序

    指定了一个元素的堆叠顺序（哪个元素应该放在前面，或后面），可以有正数或负数的堆叠顺序，具有更高堆叠顺序的元素总是在较低的堆叠顺序元素的前面。
    注意： 如果两个定位元素重叠，没有指定 z-index，最后定位在HTML代码中的元素将被显示在最前面。
    注意： z-index 进行定位元素(position:absolute, position:relative, or position:fixed)。

    auto：默认。堆叠顺序与父元素相等
    number：设置元素的堆叠顺序
    inherit：从父元素继承
