简介

CSS 指层叠样式表，使用 CSS 同时控制多重网页的样式和布局。

CSS规则主要组成：选择器以及一条或多条声明，如h1{color:red;font-size:20px;}
选择器通常是需要改变样式的元素。
每条声明由一个属性和一个值组成，属性和值使用冒号隔开。
声明总以分号 ; 结束，而声明总以大括号 {} 括起来。
CSS注释以/*xxx*/来进行注释

样式表

样式表：外部样式表（即通过link标签链接到.css样式）、内部样式表（使用<style></style>建立样式）、内联样式（直接在每个元素的属性设置style）
外部样式表链接如 <link rel="stylesheet" type="text/css" href="my.css"> 链接到my.css
注意：不要在属性值与单位之间留有空格（如："margin-left: 20 px" ），正确的写法是 "margin-left: 20px" 。

若内联、内部、外部样式表三者同时存在，则设置了一样属性的值 内联>内部>外部>浏览器默认样式，而内部没有设置的外部属性则会保留继承下来。如：
外部设置 h3{color:red;text-align:left;font-size:8pt;}
内部设置 h3{text-align:right;font-size:20pt;}
则结果是：h3{color:red;text-align:right;font-size:20pt;}

优先级：内联样式 > id选择器 > 伪类 > 属性选择器 > 类选择器 > 元素（类型）选择器 > 通用选择器（*）
注意：!important规则除外，当这个被应用在一个样式声明中，该样式声明会覆盖CSS中任何其他的声明，无论它处在声明列表中的何处
当一样时后出现的样式表设置要优先于先出现的样式表设置，在同一组属性设置中标有!important规则的优先级最大。如：p{/*两个color属性在同一组*/ color:#00f !important; /* 优先级最大 */ color:#f00; }

css的7种选择器优先级：内联样式 > id 选择器 > 类选择器 = 伪类选择器 = 属性选择器 > 标签选择器 = 伪元素选择器

选择器

id选择器：可以为标有特定id的html元素指定特定样式。
id选择器以“#”来定义，如：使用#xx{x:xxx;xx:xx;}定义，然后在html中元素属性 id="xx" 来进行使用。
注意：id属性不要以数字开头，因为在Mozilla/Firefox浏览器不起作用。

class选择器：用于描述一组元素样式。class可以在多个元素中使用，在html中以class属性表示，
类选择器以点 . 号显示，如.center{text-align:center;}
也可以指定特定的HTML元素使用class，如使所有的p元素使用class="center"让该元素文本居中。p.center {text-align:center;}
类名的第一个字符不能使用数字！它无法在 Mozilla 或 Firefox 中起作用。

分组选择器：在样式表中有很多具有相同样式的元素，此时可使用分组选择器（每个选择器使用逗号 , 隔开），如h1{color:red;}h2{color:red;}p{color:red;}，使用分组选择器之后h1,h2,p{color:red;}

嵌套选择器：适用于选择器内部的选择器的样式
p{}为所有p元素指定一个样式，这里每个p元素都会是此样式
.marked{}为所有class="marked"元素指定一个样式，如<div class="marked">这里的div内部都是是此样式
.marked p{}为所有class="marked"元素内的p元素指定一个样式，如<div class="marked"><p>这里指定的就是div里面的p元素
p.marked{}为所有的class="marked"的p元素指定一个样式，如<p class="marked">这里的p元素就是此样式

组合选择符：说明了两个选择器直接的关系。
四种组合方式：后代选择器（以空格 分隔）、子元素选择器（以大于>分隔）、相邻兄弟选择器（以加号+分隔）、普通兄弟选择器（以波浪~分隔）

后代选择器：用于选取某元素的后代元素，如：
<style>div p{color:red}</style>
<div><p>xxx</p></div>
这里的p就会是style里设置的样式

子元素选择器：相对于后代选择器，只能选择作为某元素子元素的元素，如
<style>div>p{color:yellow;}</style>
<div><p>xx</p></div>
<div><span><p>yy</p></span></div>
这里xx是style样式，而yy不会显示成style样式

相邻兄弟选择器：可选择紧接在另一元素后的元素，且二者有相同父元素。如果需要选择紧接在另一个元素后的元素，而且二者有相同的父元素，可以使用相邻兄弟选择器。如：
<style>div+p{background-color:yellow;}</style>
<div><p>xxx</p></div>
<p>xx</p>
<p>yy</p>
这里只有xx可以显示从style样式，而xxx和yy都不会有style样式

后续兄弟选择器：选取所有指定元素之后的相邻兄弟元素。如：
<style>div~p{color:yellow;}</style>
<p>xxx</p>
<div><p>xx</p></div>
<p>yy</p>
<p>yy</p>
<div><p>xx</p></div>
<p>yy</p>
这里yy都会显示style样式，而xx和xxx不会有style样式

属性选择器：如[title]{color:blue;}，这里是把包含标题 title 的所有元素变蓝

属性和值选择器：如[title=bob]{color:red;}，这里是把包含 titile="bob" 的所有元素变红

属性和值选择器 多值：
1.使用波浪 ~ 分隔属性和值：
[title~=hello] { color:blue; }
<h1 title="hello hai">Hello 嗨</h1>
<p title="student hello">Hello CSS students!</p>
<h2 title="student">Hi CSS students!</h2>
这里h1和p都会变蓝，而h2不会有效
2.使用位或 | 分隔属性和值：
[lang|=en]{color:blue;}
<p lang="en">Hello!</p>
<p lang="en-us">Hi!</p>
<p lang="us">Hi!</p>
这里前面两个都会变蓝，最后一个无效
3.属性选择器无需使用class或id形式：
<style>
  input[type="text"] { width:150px; display:block; margin-bottom:10px; background-color:yellow;}
  input[type="button"] { width:120px; margin-left:35px; display:block;}
</style>
<form method="get">
  username:<input type="text" value="name" size="20">
  password:<input type="text" value="password" size="20">
  <input type="button" value="Button">
</form>

CSS 属性选择器 ~=, |=, ^=, $=, *= 的区别：
"value 是完整单词" 类型的比较符号: ~=, |=
"拼接字符串" 类型的比较符号: *=, ^=, $=
1.attribute 属性中包含 value：
[attribute~=value] 属性中包含独立的单词为 value，如：
[title~=value]{color:red;}
<p title="max value">hello</p>
显示为红色
[attribute*=value] 属性中做字符串拆分，只要能拆出来 value 这个词就行，如：
[title~=value]{color:red;}
<p title="max value">hello</p>
<p title="minvalue">hai</p>
都会显示为红色
2.attribute 属性以 value 开头：
[attribute|=value] 属性中必须是完整且唯一的单词，或者以 - 分隔开，如：
[lang|=en]{color:red;}
<p lang="en-us">hello</p>
这里显示红色
attribute^=value] 属性的前几个字母是 value 就可以，如：
[lang^=en]{color:red;}
<p lang="ennn">hai</p>
这里显示红色
3.attribute 属性以 value 结尾：
[attibute$=value] 属性的后几个字母是 value 就可以，如：
div[class$="test"]{color:red;}
<div class="first_test">第一个 div 元素。</div>
<div class="test">第二个 div 元素。</div>
<div class="test22">第三个 div 元素。</div>
<p class="test">这是段落中的文本。</p>
这里的前两个元素显示红色，后两个元素无效

伪类和伪元素

伪类：用于添加一些选择器的特殊效果。
伪元素：用来添加一些选择器的特殊效果。

伪类的语法：
selector:pseudo-class {property:value;}
CSS类也可以使用伪类：
selector.class:pseudo-class {property:value;}

伪元素的语法：
selector:pseudo-element {property:value;}
CSS类也可以使用伪元素：
selector.class:pseudo-element {property:value;}

伪类和css类配合使用：a.red:visited {color:#FF0000;} <a class="red" href="css-syntax.html">CSS 语法</a>，链接已被访问，它会显示为红色。

anchor伪类：a:link、a:visited、a:hover、a:active

first-child 伪类：选择父元素的第一个子元素。如
p:first-child{color:blue;}
<p>xx</p>
<p>yy</p>
xx有效，yy无效
如：
p > i:first-child{color:blue;}
<<p>x<i>ii</i>xx</p>
<p>x<i>ii</i>xx<i>iiii</i>x</p>
这里只有ii显示出style样式
如：
p:first-child i{color:blue;}
<p>x<i>ii</i>xx<i>ii</i>x</p>
<p>x<i>iii</i>xx<i>iii</i>x</p>
只有ii有效，即选择器匹配所有作为元素的第一个子元素的 <p> 元素中的所有 <i> 元素

:lang 伪类：为不同的语言定义特殊的规则。如：
q:lang(no){quotes: "~" "~";}
<p>Some text <q lang="no">A quote in a paragraph</q> Some text.</p>
在q的开始和结束位置上加入了 ~ 符号

:first-line 伪元素：用于向文本的首行设置特殊样式。如：
p:first-line {color:red;}
<p>xx</p>
xx显示为红色
注意："first-line" 伪元素只能用于块级元素。

:first-letter 伪元素：用于向文本的首字母设置特殊样式，如：
p:first-letter {color:red;}
<p>xxx</p>
首个x会变红色
注意："first-letter" 伪元素只能用于块级元素。

伪元素和css类配合使用：如：
p.article:first-letter {color:#ff0000;}
<p class="article">文章段落</p>
首字母变为红色

多个伪元素：如：
p:first-letter{color:red}
p:first-line {color:blue;}
<p>你可以结合使用"first-line"和"first-letter"伪元素向文本的首行和首字母设置特殊样式。</p>
第一个字为红色，首行是蓝色

:before 伪元素：可以在元素的内容前面插入新内容。如：
h1:before {content:url(smiley.gif);}
<h1>xx</h1>
在xx文本前面有个.gif图像

:after 伪元素：可以在元素的内容之后插入新内容。如：
h1:after {content:url(smiley.gif);}
<h1>xx</h1>
在xx文本后面有个.gif图像

链接

四个链接状态：
a:link：正常，未访问过的链接
a:visited：用户已访问过的链接
a:hover：用户鼠标发在链接时
a:active：链接被点击的那刻
注意：当多个样式设置时，hover必须在link和visited后面，而active必须在hover后面

background：背景设置

优先级为 color > image > position/size > repeat > origin > clip > attachment > 默认/继承。整个背景简写如background:#ffffff url('img_tree.png') no-repeat fixed right top;

CSS3中可以给不同的图片设置多个不同的属性，如：background: url(img_flwr.gif) right bottom no-repeat, url(paper.gif) left top repeat;

background-attachment：背景是否滚动

scroll：背景图随页面的滚动而滚动，默认
fixed：背景图不随页面滚动而滚动
local：背景图随元素内容的滚动二滚动
initial：默认值
inherit：从父元素继承

background-color：背景色

1.名称：red、green、yellow、gray
2.RGB值：rgb(255,0,0)
3.十六进制：#ffffff
transparent：背景色透明，默认值
inherit：从父元素继承

此背景色还可以用于标识文字的背景色，相当于标亮

background-image：背景图

背景图片设置：url('xx.png')
none：无背景图，默认
linear-gradient()：创建一个线性渐变图（从上到下）
radial-gradient()：用径向渐变创建图像（中间到边缘）
repeating-linear-gradient()：创建重复的线性渐变图
repeating-radial-gradient()：创建重复的径向渐变图
inherit：从父元素继承

    CSS3 是最新的 CSS 标准，完全向后兼容。

    border-radius：创建圆角

    是一个最多可指定四个 border -*- radius 属性的复合属性。

    注意: 每个半径的四个值的顺序是：左上角，右上角，右下角，左下角。如果省略左下角，右上角是相同的。如果省略右下角，左上角是相同的。如果省略右上角，左上角是相同的。

    四个值为左上、右上、右下、左上角；三个值为左上（右上/左下）右下；两个值为（左上/右下）（右上/左下），一个值为全一样

    length：定义弯道的形状
    百分比：使用%定义角落的形状

    border-top-left-radius：左上角的弧度

    两个长度或百分比值定义了椭圆的四分之一外边框的边缘角落的形状。第一个值是水平半径，第二个是垂直半径。如果省略第二个值，它是从第一个复制。如果任一长度为零，角落里是方的，不圆润。水平半径的百分比是指边界框的宽度，而垂直半径的百分比是指边界框的高度。

    length：定义左上角的形状
    百分比：使用%定义左上角的形状。

    border-top-right-radius：右上角的弧度

    两个长度或百分比值定义了椭圆的四分之一外边框的边缘角落的形状。第一个值是水平半径，第二个是垂直半径。如果省略第二个值，它是从第一个复制。如果任一长度为零，角落里是方的，不圆润。水平半径的百分比是指边界框的宽度，而垂直半径的百分比是指边界框的高度。

    length：定义右上角的形状
    百分比：使用%定义右上角的形状。

    border-bottom-right-radius：右下角的弧度

    两个长度或百分比值定义了椭圆的四分之一外边框的边缘角落的形状。第一个值是水平半径，第二个是垂直半径。如果省略第二个值，它是从第一个复制。如果任一长度为零，角落里是方的，不圆润。水平半径的百分比是指边界框的宽度，而垂直半径的百分比是指边界框的高度。

    length：定义右下角的形状
    百分比：使用%定义右下角的形状。

    border-bottom-left-radius：左下角的弧度

    两个长度或百分比值定义了椭圆的四分之一外边框的边缘角落的形状。第一个值是水平半径，第二个是垂直半径。如果省略第二个值，它是从第一个复制。如果任一长度为零，角落里是方的，不圆润。水平半径的百分比是指边界框的宽度，而垂直半径的百分比是指边界框的高度。

    length：定义左下角的形状
    百分比：使用%定义左下角的形状。

    顺序：h-shadow > v-shadow blur > spread > color > inset

    h-shadow 必需的。水平阴影的位置。允许负值
    v-shadow：必需的。垂直阴影的位置。允许负值
    blur：可选。模糊距离
    spread：可选。阴影的大小
    color：可选。阴影的颜色。在CSS颜色值寻找颜色值的完整列表
    inset：可选。从外层的阴影（开始时）改变阴影内侧阴影

    border-image：使用图像创建一个边框

    顺序： border-image-source > border-image-slice > border-image-width > border-image-outset > border-image-repeat|initial|inherit

    -webkit-border-image：safari浏览器支持
    -o-border-image：opera浏览器支持

    border-image-source：指定要用于绘制边框的图像的位置

    none：没有图像被使用
    image：边框使用图像的路径

    border-image-slice：图像边界向内偏移

    此属性指定顶部，右，底部，左边缘的图像向内偏移，分为九个区域：四个角，四边和中间。图像中间部分将被丢弃（完全透明的处理），除非填写关键字。如果省略第四个数字/百分比，它和第二个相同的。如果也省略了第三个，它和第一个是相同的。如果也省略了第二个，它和第一个是相同的。

    number：数字表示图像的像素（位图图像）或向量的坐标（如果图像是矢量图像）
    百分比：百分比图像的大小是相对的：水平偏移图像的宽度，垂直偏移图像的高度
    fill：保留图像的中间部分

    border-image-width：图像边界的宽度

    4个值指定用于把border图像区域分为九个部分。他们代表上，右，底部，左，两侧向内距离。如果第四个值被省略，它和第二个是相同的。如果也省略了第三个，它和第一个是相同的。如果也省略了第二个，它和第一个是相同的。负值是不允许的。

    number：表示相应的border-width 的倍数
    百分比：边界图像区域的大小：横向偏移的宽度的面积，垂直偏移的高度的面积
    auto：如果指定了，宽度是相应的image slice的内在宽度或高度

    border-image-outset：用于指定在边框外部绘制 border-image-area 的量

    用于指定在边框外部绘制 border-image-area 的量。包括上下部和左右部分。如果第四个值被省略，它和第二个是相同的。如果也省略了第三个，它和第一个是相同的。如果也省略了第二个，它和第一个是相同的。不允许border-im-outset拥有负值。

    length：设置边框图像与边框（border-image）的距离，默认为0
    number：代表相应的 border-width 的倍数

    border-image-repeat：用于设置图像边界是否应重复\拉伸\铺满

    stretch：默认值。拉伸图像来填充区域
    repeat：平铺（repeated）图像来填充区域，重复
    round：类似 repeat 值。如果无法完整平铺所有图像，则对图像进行缩放以适应区域
    space：类似 repeat 值。如果无法完整平铺所有图像，扩展空间会分布在图像周围
    initial：将此属性设置为默认值
    inherit：从父元素继承

    background-clip：背景图的绘画区域

    border-box：背景绘制在边框方框内（剪切成边框方框），默认值
    padding-box：背景绘制在衬距方框内（剪切成衬距方框）
    content-box：背景绘制在内容方框内（剪切成内容方框）

    background-origin：背景图的定位区域

    padding-box：背景图填充框的相对位置
    border-box：背景图边界框的相对位置
    content-box：背景图的相对位置的内容框

    background-size：背景图大小

    length：背景图高宽，第一个值为宽度，第二个值为高度，若只给一个值则第二个值默认auto（自动）
    percentage：百分比，第一个宽，第二个高，若只设置一个，则第二个默认auto，如100% 100%
    cover：保持图像的纵横比并将图像缩放成将完全覆盖背景定位区域的最小大小
    contain：保持图像的纵横比并将图像缩放成将适合背景定位区域的最大大小

    CSS3以前，背景图像大小由图像的实际大小决定。CSS3中可以指定背景图片，让我们重新在不同的环境中指定背景图片的大小。您可以指定像素或百分比大小。你指定的大小是相对于父元素的宽度和高度的百分比的大小。

    Gradients：渐变

    线性渐变和径向渐变
    线性渐变：向下/向上/向左/向右/对角方向
    径向渐变：由中心定义。
    线性渐变相关属性：background-image

    线性渐变，你必须至少定义两种颜色节点。颜色节点即你想要呈现平稳过渡的颜色。同时，你也可以设置一个起点和一个方向（或一个角度）。

    线性语法：background-image: linear-gradient(direction, color-stop1, color-stop2, ...);
    从上到下：background-image: linear-gradient(red, blue);
    从左到右：height: 200px;background-image: linear-gradient(to right, red , yellow);
    对角：过指定水平和垂直的起始位置来制作一个对角渐变。height: 200px;background-image: linear-gradient(to bottom right, red, yellow);左上到右下

    加入角度语法：background-image: linear-gradient(angle, color-stop1, color-stop2);
    角度是指水平线和渐变线之间的角度，逆时针方向计算。换句话说，0deg 将创建一个从下到上的渐变，90deg 将创建一个从左到右的渐变。
    background-image: linear-gradient(-90deg, red, yellow);

    多颜色：background-image: linear-gradient(red, yellow, green);
    background-image: linear-gradient(to right, red,orange,yellow,green,blue,indigo,violet);带有彩虹颜色和文本的线性渐变

    透明度：background-image: linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1));从左边开始的线性渐变。起点是完全透明，慢慢过渡到完全不透明的红色
    为了添加透明度，我们使用 rgba() 函数来定义颜色节点。rgba() 函数中的最后一个参数可以是从 0 到 1 的值，它定义了颜色的透明度：0 表示完全透明，1 表示完全不透明。

    重复的线性渐变：background-image: repeating-linear-gradient(red, yellow 10%, green 20%);

    径向渐变：径向渐变由它的中心定义。为了创建一个径向渐变，你也必须至少定义两种颜色节点。颜色节点即你想要呈现平稳过渡的颜色。同时，你也可以指定渐变的中心、形状（圆形或椭圆形）、大小。默认情况下，渐变的中心是 center（表示在中心点），渐变的形状是 ellipse（表示椭圆形），渐变的大小是 farthest-corner（表示到最远的角落）。

    径向语法：background-image: radial-gradient(shape size at position, start-color, ..., last-color);

    颜色节点均匀分布的径向渐变：background-image: radial-gradient(red, yellow, green);

    颜色节点不均匀分布的径向渐变：background-image: radial-gradient(red 5%, yellow 15%, green 60%);

    设置形状：shape 参数定义了形状。它可以是值 circle 或 ellipse。其中，circle 表示圆形，ellipse 表示椭圆形。默认值是 ellipse。如 background-image: radial-gradient(circle, red, yellow, green);

    不同尺寸大小关键字的使用：closest-side、farthest-side、closest-corner、farthest-corner
    带有不同尺寸大小关键字的径向渐变
    background-image: radial-gradient(closest-side at 60% 55%, red, yellow, black);
    background-image: radial-gradient(farthest-side at 60% 55%, red, yellow, black);

    重复的径向渐变：background-image: repeating-radial-gradient(red, yellow 10%, green 15%);

    text-shadow：文字阴影

    优先级：h-shadow > v-shadow > blur > colo

    h-shadow：必需。水平阴影的位置。允许负值。
    v-shadow：必需。垂直阴影的位置。允许负值。
    blur：可选。模糊的距离。
    color：可选。阴影的颜色

    box-shadow：添加阴影

    优先级：水平阴影 > 垂直阴影 > 模糊的距离（值越大越模糊） > 阴影的颜色

    h-shadow（水平阴影）> v-shadow（垂直阴影）> blur（模糊）> spread（阴影尺寸）> color（颜色）>insect（外阴影转到内阴影）

    设置一个或多个下拉阴影的框。
    把一个或多个下拉阴影添加到框上。该属性是一个用逗号分隔阴影的列表，每个阴影由 2-4 个长度值、一个可选的颜色值和一个可选的 inset 关键字来规定。省略长度的值是 0。

    h-shadow：必需。水平阴影的位置。允许负值
    v-shadow：必需。垂直阴影的位置。允许负值
    blur：可选。模糊距离
    spread：可选。阴影的尺寸
    color：可选。阴影的颜色。请参阅 CSS 颜色值
    inset：可选。将外部阴影 (outset) 改为内部阴影

    word-wrap：允许对长的不可分割的单词进行分割并换行到下一行

    允许长的内容可以自动换行。

    normal：只在允许的断字点换行（浏览器保持默认处理）
    break-word：在长单词或 URL 地址内部进行换行。

    word-break：规定非中日韩文本的换行规则

    normal：使用浏览器默认的换行规则
    break-all：允许在单词内换行
    keep-all：只能在半角空格或连字符处换行

    hanging-punctuation：标点字符是否位于线框之外

    指定一个标点符号是否会在启动或在结束时文本行框以外。

    none：不在文本整行的开头还是结尾的行框之外放置标签符号
    first：标点附着在首行开始边缘之外
    last：标点附着在首行结尾边缘之外
    allow-end
    force-end

    punctuation-trim：是否对标点字符进行修剪

    指定是否应修饰，当它出现在一条线，或相邻的另一个全形标点符号字符开头或结束标点字符。

    none 请勿修剪打开或关闭标点符号
    start：在每一行的开头放置开头标点符号
    end：在每一行的末尾修剪结束标点符号
    allow-end：如果另有不适合之前的理由，修剪每行末尾的结束标点符号
    adjacent：若以前相邻字符修剪开口punctuation是一个全形开口，中间或结束标点符号，或表意文字空间。修剪结束标点符号，如果下一个相邻的字符是一个全形关闭或中间punctuation，或表意文字空间

    text-align-last：如何对齐最后一行或紧挨着强制换行符之前的行

    规定如何对齐文本的最后一行。
    注意：text-align-last 属性只有在 text-align 属性设置为 "justify" 时才起作用。

    auto：默认值。最后一行被调整，并向左对齐
    left：最后一行向左对齐
    right：最后一行向右对齐
    center：最后一行居中对齐
    justify：最后一行被调整为两端对齐
    start：最后一行在行开头对齐（如果 text-direction 是从左到右，则向左对齐；如果 text-direction 是从右到左，则向右对齐）
    end：最后一行在行末尾对齐（如果 text-direction 是从左到右，则向右对齐；如果 text-direction 是从右到左，则向左对齐）
    initial：设置该属性为它的默认值
    inherit：从父元素继承

    text-emphasis：向元素的文本应用重点标记以及重点标记的前景色

    优先级：text-emphasis-style > text-emphasis-color
    text-emphasis-style：向元素的文本应用重点标记
    text-emphasis-color：定义重点标记的前景色

    text-justify：规定当 text-align 设置为 "justify" 时所使用的对齐方法

    指定文本对齐设置为"justify"的理据。此属性指定应怎样对齐文本以及对齐间距。

    auto：浏览器决定齐行算法
    none：禁用齐行
    inter-word：增加/减少单词间的间隔
    inter-ideograph：用表意文本来排齐内容
    inter-cluster：只对不包含内部单词间隔的内容（比如亚洲语系）进行排齐
    distribute：类似报纸版面，除了在东亚语系中最后一行是不齐行的
    kashida：通过拉伸字符来排齐内容。

    text-outline：文本的轮廓

    优先级：thickness > blur > color

    thickness：必需。轮廓的粗细
    blur：可选。轮廓的模糊半径
    color：必需。轮廓的颜色

    text-overflow：如何显示溢出内容

    指定当文本溢出包含它的元素，应该发生什么。

    clip：修剪文本
    ellipsis：显示省略符号来代表被修剪的文本
    string：使用给定的字符串来代表被修剪的文本

    text-shadow：设置文本阴影

    优先级：h-shadow > v-shadow > blur > color

    h-shadow：必需。水平阴影的位置。允许负值
    v-shadow：必需。垂直阴影的位置。允许负值
    blur：可选。模糊的距离
    color：可选。阴影的颜色

    text-wrap：文本的换行规则

    normal：只在允许的换行点进行换行
    none：不换行。元素无法容纳的文本会溢出
    unrestricted：在任意两个字符间换行
    suppress：压缩元素中的换行。浏览器只在行中没有其他有效换行点时进行换行
