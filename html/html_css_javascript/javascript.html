字面量：固定值。
数字字面量的整数或小数或科学计数；如var=123e4;
字符串字面量的单双引号括起来；如var value='hello "bob"';
表达式字面量的加减乘除等；如var a=b+c;
数组字面量的中括号[]括起来；如var arr=new Array();arr[0]="a";arr[1]="b";或var arr=new Array("a","b");或var arr=["a","b"];
对象字面量的花括号{}括起来；如var obj={first:"a",last:"b"};value=obj.first;//value=obj["first"];
函数字面量的function定义一个函数；如function my(a,b){return a+b;}

变量：用于存储数据值，使用关键字var定义变量（多个变量可使用逗号 , 分隔），使用等号为变量赋值（若未赋值则默认是undefined），变量必须以字母开头（也可用$和_符号开头，但不推荐），变量前面用过后面再重新声明则前面的值依旧在，对大小写敏感。如 x=6; var x;
变量通常是可变的，而字面量是一个恒定的值。
声明变量类型：

var c = new Boolean;
var d = new Array;
var e = new Object;

变量提升：函数声明和变量声明总是会被解释器悄悄地被"提升"到方法体的最顶部。如变量可先使用后声明，但是只有声明的变量会提升，而初始化不会提升，也就是若是前面先使用后面再声明并初始化的话，输出的值可能不是初始化的值二十undefined，也就是声明提升了，初始化不会提升，所以变量是一个未定义的变量

对象

JavaScript 对象是拥有属性和方法的数据。是变量的容器，内部以键值对出现，分号前面的是变量，分号后面的是值

对象属性：键值对写法为name:value，在JavaScript对象中通常称为对象属性。
可使用两种方式访问属性：1.obj.name; //一点来获取；2.obj["name"]; //以中括号获取

自定义对象方法：
<p id="demo"></p>
<script>
var person = {
  firstName: "John",
  lastName : "Doe",
  id : 5566,
  fullName : function(){ //自定义对象函数
    return this.firstName + " " + this.lastName;} //返回变量组合的字符串
};
document.getElementById("demo").innerHTML = person.fullName(); //赋值自定义对象方法，若没加括号，则回去的是fullName属性，返回的是function整个函数

函数

函数是由事件驱动的或者当它被调用时执行的可重复使用的代码块。可在某事件发生时直接调用函数（如点击按钮事件）

函数语法：包括在花括号内的代码块，前面使用关键字function，即function name(){xxx;return xx;}
参数：在括号里面，多个参数由逗号 , 隔开
返回值：可有可无，当函数遇到此关键字时将不再执行后面语句，并返回指定的值（也可直接不带值返回，返回return）

如果您把值赋给尚未声明的变量，该变量将被自动作为 window 的一个属性。如name="bob";//将声明window的一个属性name
非严格模式下给未声明变量赋值创建的全局变量，是全局对象的可配置属性，可以删除。
var var1 = 1; // 不可配置全局属性
var2 = 2; // 没有使用 var 声明，可配置全局属性
console.log(this.var1); // 1
console.log(window.var1); // 1
console.log(window.var2); // 2
delete var1; // false 无法删除
console.log(var1); //1
delete var2;
console.log(delete var2); // true
console.log(var2); // 已经删除 报错变量未定义

作用域

可访问变量的集合，即在那个地方哪个变量是全局的还是局部的
局部作用域：在函数内声明的变量只能在函数内使用，所以不同的函数可以使用相同的变量名（和C类似）
全局作用域：变量在函数外定义即全局变量，变量在函数内没有声明（没有var关键字）则改变了也是全局变量，网页中所有脚本和函数均可使用

在 HTML 中, 全局变量是 window 对象: 所有数据变量都属于 window 对象。

正则表达式

正则表达式，使用单个字符串来描述、匹配一系列符合某个句法规则的字符串搜索模式。搜索模式可用于文本搜索和文本替换。

语法：/正则表达式主体/修饰符(可选)，如var name = /bob/i（ /bob/i为正则表达式，bob为主体，用于检索，i是一个修饰符，搜索不区分大小写）
修饰符：i 不区分大小写；g 全局匹配（查找所有匹配而非在找到第一个匹配后停止）；m 执行多行匹配
正则表达式：1.[]搜索某个范围内字符（[abc]查找括号内任意字符），[0-9]（查找从0到9数字），(x|y)（查找任何以|分隔的选项）；2.元字符特殊含义，\d（查找数字）。\s（查找空白字符），\b（匹配单词边界），\uxxxx（查找以十六进制规定的Unicode字符）；3.量词，n+（匹配任意包含至少一个n的字符串），n*（匹配任意包含零一或多个n的字符串），n?（匹配任意包含零个或一个n的字符串）

方法：
1.search()方法由于减少字符串中指定的子字符串，或检索与正则表达式匹配的子字符串，并返回子串的起始位置。如 var str = "Welcome To You!"; var n = str.search(/you/i);var m = str.search("you");
2.replace()方法用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达匹配的子串。如 var str = "Welcome To You!"; var n = str.replace(/you/i,"me"); var m = str.replace("you","me");
3.test()方法检测一个字符串是否匹配某个模式，若字符串包含匹配文本返回true，否则false，如 var patt = /e/; patt.test("this is test funtion.");//两语句可合并成 /e/.test("this is test funtion.")
4.exec()方法用于检索字符串中的正则表达式的匹配，如 /e/.exec("this is test funtion.")

错误调试

try：测试代码块错误；catch：处理错误；throw：创建自定义错误；finally：在try和catch语句后，无论是否触发异常都会执行，类似于python的异常处理

语法：try{...;//异常抛出}catch(e){...;异常捕获与处理}finally{...;//结束处理}

throw：语法 throw exception，创建或抛出异常，如果把 throw 与 try 和 catch 一起使用，那么您能够控制程序流，并生成自定义的错误消息。如：
try{
  if(x=="") throw "值为空";
  if(isNaN(x))throw "不是数字";
}
catch(e){
  message.innerHTML="错误：" + e;
}

console.log()方法：可在调试窗口打印 JavaScript 值（浏览器启动调试工具一般按 F12 键，然后选中console）

debugger关键字：用于停止执行 JavaScript，并调用调试函数。这个关键字与在调试工具中设置断点的效果是一样的。

表单验证

判断是否有输入，没有则弹窗提示：
<script>
function validateForm() {
  var x = document.forms["myForm"]["fname"].value;
  if (x == null || x == "") {
    alert("需要输入名字。");
    return false;}}
</script>
<form name="myForm" action="demo_form.php" onsubmit="return validateForm()" method="post">
名字: <input type="text" name="fname">
<input type="submit" value="提交">
</form>
此为JavaScript实现，下面为单纯的html实现（fname值为空，required属性会阻止表单提交）：
<form action="demo_form.php" method="post">
  <input type="text" name="fname" required="required">
  <input type="submit" value="提交">
</form>

rangeUnderflow：设置为 true, 如果元素的值小于它的最小值
stepMismatch：设置为 true, 如果元素的值不是按照规定的 step 属性设置
tooLong 设置为：true, 如果元素的值超过了 maxLength 属性设置的长度
typeMismatch：设置为 true, 如果元素的值不是预期相匹配的类型
valueMissing：设置为 true，如果元素 (required 属性) 没有值
valid：设置为 true，如果元素的值是合法的

this、let、const

this：不是固定不变的，它会随着执行环境的改变而改变。1.在方法中表示该方法所属对象；2.单独使用表示全局对象(在浏览器中，window就是该全局对象)；3.在函数中表示全局变量（"use strict"严格模式下标为未定义）；4.在事件中表示接收事件的元素；

this示例：
1.方法
<script>

</script>
