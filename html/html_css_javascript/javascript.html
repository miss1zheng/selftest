外部引用，保存在.js文件中，然后在html中使用<script src="myScript.js"></script>获取

字面量：固定值。
数字字面量的整数或小数或科学计数；如var=123e4;
字符串字面量的单双引号括起来；如var value='hello "bob"';
表达式字面量的加减乘除等；如var a=b+c;
数组字面量的中括号[]括起来；如var arr=new Array();arr[0]="a";arr[1]="b";或var arr=new Array("a","b");或var arr=["a","b"];
对象字面量的花括号{}括起来；如var obj={first:"a",last:"b"};value=obj.first;//value=obj["first"];
函数字面量的function定义一个函数；如function my(a,b){return a+b;}

变量：用于存储数据值，使用关键字var定义变量（多个变量可使用逗号 , 分隔），使用等号为变量赋值（若未赋值则默认是undefined），变量必须以字母开头（也可用$和_符号开头，但不推荐），变量前面用过后面再重新声明则前面的值依旧在，对大小写敏感。如var x; x=6
变量通常是可变的，而字面量是一个恒定的值。
声明变量类型：
var a = new String;
var b = new Number;
var c = new Boolean;
var d = new Array;
var e = new Object;

对象

JavaScript 对象是拥有属性和方法的数据。是变量的容器，内部以键值对出现，分号前面的是变量，分号后面的是值

对象属性：键值对写法为name:value，在JavaScript对象中通常称为对象属性。
可使用两种方式访问属性：1.obj.name; //一点来获取；2.obj["name"]; //以中括号获取

自定义对象方法：
<p id="demo"></p>
<script>
var person = {
  firstName: "John",
  lastName : "Doe",
  id : 5566,
  fullName : function(){ //自定义对象函数
    return this.firstName + " " + this.lastName;} //返回变量组合的字符串
};
document.getElementById("demo").innerHTML = person.fullName(); //赋值自定义对象方法，若没加括号，则回去的是fullName属性，返回的是function整个函数
</script>

函数

函数是由事件驱动的或者当它被调用时执行的可重复使用的代码块。可在某事件发生时直接调用函数（如点击按钮事件）

函数语法：包括在花括号内的代码块，前面使用关键字function，即function name(){xxx;return xx;}
参数：在括号里面，多个参数由逗号 , 隔开
返回值：可有可无，当函数遇到此关键字时将不再执行后面语句，并返回指定的值（也可直接不带值返回，返回return）

如果您把值赋给尚未声明的变量，该变量将被自动作为 window 的一个属性。如name="bob";//将声明window的一个属性name
非严格模式下给未声明变量赋值创建的全局变量，是全局对象的可配置属性，可以删除。
var var1 = 1; // 不可配置全局属性
var2 = 2; // 没有使用 var 声明，可配置全局属性
console.log(this.var1); // 1
console.log(window.var1); // 1
console.log(window.var2); // 2
delete var1; // false 无法删除
console.log(var1); //1
delete var2;
console.log(delete var2); // true
console.log(var2); // 已经删除 报错变量未定义

作用域

可访问变量的集合，即在那个地方哪个变量是全局的还是局部的
局部作用域：在函数内声明的变量只能在函数内使用，所以不同的函数可以使用相同的变量名（和C类似）
全局作用域：变量在函数外定义即全局变量，变量在函数内没有声明（没有var关键字）则改变了也是全局变量，网页中所有脚本和函数均可使用

在 HTML 中, 全局变量是 window 对象: 所有数据变量都属于 window 对象。

字符串

用于存储和处理文本。使用单引号或双引号。使用索引（即中括号[]）位置来访问字符串中的每个字符，从0开始。在字符串中使用引号，字符串中的引号不要与字符串的引号相同

字符串转义字符：\'、\"、\\、\n、\r、\t、\b、\f等（对应于单双引号、反斜杠、换行、回车、制表符tab、退格符、换页符）

内置属性length计算字符串长度；typeof返回变量类型

字符串为对象的情况，var name=new String("bob");//不建议创建string对象
