简介

脚本语言，和html、css一起用于web浏览器

对事件做出反应：如<button type="button" onclick="alert('欢迎!')">点我!</button>
点击后会出现警告，其中alert() 函数是发出警告，一般用于代码测试。
onclick 事件在点击时发生的事件

javascript脚本在html中除了外部引用，在html内部使用必须位与<script> 与 </script> 标签之间。
一般是将JavaScript放在head里面或者放在页面底部，这样放在一个位置不会干扰页面内容
外部引用，保存在.js文件中，然后在html中使用<script src="myScript.js"></script>获取

浏览器中JavaScript的调试：在chrome浏览器中按下F12或右击页面，“检查”开启“开发者工具”，然后进入Coonsole窗口调试界面

字面量：固定值。
数字字面量的整数或小数或科学计数；如var=123e4;
字符串字面量的单双引号括起来；如var value='hello "bob"';
表达式字面量的加减乘除等；如var a=b+c;
数组字面量的中括号[]括起来；如var arr=new Array();arr[0]="a";arr[1]="b";或var arr=new Array("a","b");或var arr=["a","b"];
对象字面量的花括号{}括起来；如var obj={first:"a",last:"b"};value=obj.first;//value=obj["first"];
函数字面量的function定义一个函数；如function my(a,b){return a+b;}

变量：用于存储数据值，使用关键字var定义变量（多个变量可使用逗号 , 分隔），使用等号为变量赋值（若未赋值则默认是undefined），变量必须以字母开头（也可用$和_符号开头，但不推荐），变量前面用过后面再重新声明则前面的值依旧在，对大小写敏感。如 x=6; var x;
变量通常是可变的，而字面量是一个恒定的值。
声明变量类型：
var a = new String;
var b = new Number;
var c = new Boolean;
var d = new Array;
var e = new Object;
注意：变量声明时如果不使用 var 关键字，那么它就是一个全局变量，即便它在函数内定义。

变量提升：函数声明和变量声明总是会被解释器悄悄地被"提升"到方法体的最顶部。如变量可先使用后声明，但是只有声明的变量会提升，而初始化不会提升，也就是若是前面先使用后面再声明并初始化的话，输出的值可能不是初始化的值二十undefined，也就是声明提升了，初始化不会提升，所以变量是一个未定义的变量

对象

JavaScript 对象是拥有属性和方法的数据。是变量的容器，内部以键值对出现，分号前面的是变量，分号后面的是值

对象属性：键值对写法为name:value，在JavaScript对象中通常称为对象属性。
可使用两种方式访问属性：1.obj.name; //一点来获取；2.obj["name"]; //以中括号获取

自定义对象方法：
<p id="demo"></p>
<script>
var person = {
  firstName: "John",
  lastName : "Doe",
  id : 5566,
  fullName : function(){ //自定义对象函数
    return this.firstName + " " + this.lastName;} //返回变量组合的字符串
};
document.getElementById("demo").innerHTML = person.fullName(); //赋值自定义对象方法，若没加括号，则回去的是fullName属性，返回的是function整个函数
</script>

函数

函数是由事件驱动的或者当它被调用时执行的可重复使用的代码块。可在某事件发生时直接调用函数（如点击按钮事件）

函数语法：包括在花括号内的代码块，前面使用关键字function，即function name(){xxx;return xx;}
参数：在括号里面，多个参数由逗号 , 隔开
返回值：可有可无，当函数遇到此关键字时将不再执行后面语句，并返回指定的值（也可直接不带值返回，返回return）

如果您把值赋给尚未声明的变量，该变量将被自动作为 window 的一个属性。如name="bob";//将声明window的一个属性name
非严格模式下给未声明变量赋值创建的全局变量，是全局对象的可配置属性，可以删除。
var var1 = 1; // 不可配置全局属性
var2 = 2; // 没有使用 var 声明，可配置全局属性
console.log(this.var1); // 1
console.log(window.var1); // 1
console.log(window.var2); // 2
delete var1; // false 无法删除
console.log(var1); //1
delete var2;
console.log(delete var2); // true
console.log(var2); // 已经删除 报错变量未定义

函数：语法为function functionName(parameters) { 执行的代码;}，函数可以重复使用，函数声明后不会立即执行，会在我们需要的时候调用到。JavaScript语句写在花括号{}内
函数可以通过一个表达式定义，函数表达式（实际上是一个匿名函数）可以存储在变量中，如：var x = function (a, b) {return a * b};
在函数表达式存储在变量后，变量也可作为一个函数使用，如：var x = function (a, b) {return a * b}; var z = x(4, 3);
构造函数：如果函数调用前使用了 new 关键字, 则是调用了构造函数。如 var myFunction = new Function("a", "b", "return a * b"); var x = myFunction(4, 3);
函数可以在声明之前调用，即函数提升

自调用函数（实际上是一个匿名自我调用的函数）：如果表达式后面紧跟 () ，则会自动调用。通过添加括号，来说明它是一个函数表达式，如：
(function () {
  var x = "Hello!!"; // 我将调用自己
})();

箭头函数：语法为
1.(参数1, 参数2, …, 参数N) => { 函数声明 }
2.(参数1, 参数2, …, 参数N) => 表达式(单一)//相当于：(参数1, 参数2, …, 参数N) =>{ return 表达式; }
当只有一个参数时，圆括号是可选的，(单一参数) => {函数声明}//单一参数 => {函数声明}
没有参数的函数应该写成一对圆括号，() => {函数声明}

函数显式参数：语法 functionName(parameter1, parameter2, parameter3) {// 要执行的代码……;};；函数显式参数在函数定义时列出。函数定义显式参数时没有指定数据类型。

函数隐式参数：函数隐式参数在函数调用时传递给函数真正的值。函数对隐式参数没有进行类型检测。函数对隐式参数的个数没有进行检测。

作用域

可访问变量的集合，即在那个地方哪个变量是全局的还是局部的
局部作用域：在函数内声明的变量只能在函数内使用，所以不同的函数可以使用相同的变量名（和C类似）
全局作用域：变量在函数外定义即全局变量，变量在函数内没有声明（没有var关键字）则改变了也是全局变量，网页中所有脚本和函数均可使用

在 HTML 中, 全局变量是 window 对象: 所有数据变量都属于 window 对象。

RegExp 正则表达式对象

正则表达式：描述字符模式的对象，用于对字符串模式匹配及检索替换，是对字符串执行模式匹配的强大工具。。

语法：var patt=new RegExp(pattern,modifiers); 或 var patt=/pattern/modifiers;
模式描述了一个表达式模型。修饰符(modifiers)描述了检索是否是全局，区分大小写，多行匹配等。

注意：当使用构造函数创造正则对象时，需要常规的字符转义规则（在前面加反斜杠 \）。如：var re = new RegExp("\\w+"); 等价于 var re = /\w+/;

修饰符：用于执行不区分大小写和全文的搜索。
  i：修饰符是用来执行不区分大小写的匹配。如 var str = "Hello"; var patt1 = /hello/i;
  g：修饰符是用于执行全文的搜索（而不是在找到第一个就停止查找,而是找到所有的匹配）。如 var str="this is project."; var patt1=/is/g;
  m：执行多行匹配
全文查找和不区分大小写搜索 "is"，如 var str="This Is Project."; var patt1=/is/gi;

正则表达式

正则表达式，使用单个字符串来描述、匹配一系列符合某个句法规则的字符串搜索模式。搜索模式可用于文本搜索和文本替换。

语法：/正则表达式主体/修饰符(可选)，如var name = /bob/i（ /bob/i为正则表达式，bob为主体，用于检索，i是一个修饰符，搜索不区分大小写）
修饰符：i 不区分大小写；g 全局匹配（查找所有匹配而非在找到第一个匹配后停止）；m 执行多行匹配
正则表达式：1.[]搜索某个范围内字符（[abc]查找括号内任意字符），[0-9]（查找从0到9数字），(x|y)（查找任何以|分隔的选项）；2.元字符特殊含义，\d（查找数字）。\s（查找空白字符），\b（匹配单词边界），\uxxxx（查找以十六进制规定的Unicode字符）；3.量词，n+（匹配任意包含至少一个n的字符串），n*（匹配任意包含零一或多个n的字符串），n?（匹配任意包含零个或一个n的字符串）

方法：
1.search()方法由于减少字符串中指定的子字符串，或检索与正则表达式匹配的子字符串，并返回子串的起始位置。如 var str = "Welcome To You!"; var n = str.search(/you/i);var m = str.search("you");
2.replace()方法用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达匹配的子串。如 var str = "Welcome To You!"; var n = str.replace(/you/i,"me"); var m = str.replace("you","me");
3.test()方法检测一个字符串是否匹配某个模式，若字符串包含匹配文本返回true，否则false，如 var patt = /e/; patt.test("this is test funtion.");//两语句可合并成 /e/.test("this is test funtion.")
4.exec()方法用于检索字符串中的正则表达式的匹配，如 /e/.exec("this is test funtion.")

计时事件

计时事件：在一个设定的时间间隔之后来执行代码，而不是在函数被调用后立即执行。

setInterval()：间隔指定的毫秒数不停地执行指定的代码。可以不使用 window 前缀，直接使用函数 setInterval()。
语法：window.setInterval("javascript function",milliseconds);
  参1：函数
  参2：间隔的毫秒数

clearInterval()：停止setInterval()方法执行的函数代码。window.clearInterval() 方法可以不使用window前缀，直接使用函数clearInterval()。要使用 clearInterval() 方法, 在创建计时方法时你必须使用全局变量
语法：window.clearInterval(intervalVariable)

setTimeout()：在指定的毫秒数后执行指定代码。
myVar= window.setTimeout("javascript function", milliseconds);
  参1：含有JavaScript语句的字符串，可能是函数或对函数的调用
  参2：从当前起多数毫秒后执行第一个参数
例如：setTimeout(function(){alert("Hello")},3000);

clearTimeout()：用于停止执行setTimeout()方法的函数代码。window.clearTimeout() 方法可以不使用window 前缀。要使用clearTimeout() 方法, 你必须在创建超时方法中（setTimeout）使用全局变量。如果函数还未被执行，你可以使用 clearTimeout() 方法来停止执行函数代码。
语法：window.clearTimeout(timeoutVariable)

Cookie

Cookie：用于存储 web 页面的用户信息。当 web 服务器向浏览器发送 web 页面时，在连接关闭后，服务端不会记录用户的信息。所以直接本地存储信息。

Cookie 以名/值对形式存储。当浏览器从服务器上请求 web 页面时， 属于该页面的 cookie 会被添加到该请求中。服务端通过这种方式来获取用户的信息。

document.cookie 属性：创建 、读取、及删除 cookie。

读取：将以字符串的方式返回所有的 cookie。如 var x = document.cookie; document.cookie

修改：修改 cookie 类似于创建 cookie，旧的 cookie 将被覆盖。如 document.cookie="username=John Smith; expires=Thu, 18 Dec 2043 12:00:00 GMT; path=/";

删除：只需要设置 expires 参数为以前的时间即可。注意删除时不必指定 cookie 的值。如 document.cookie = "username=; expires=Thu, 01 Jan 1970 00:00:00 GMT";

** Cookie 值 ** 设置 cookie 值：
function setCookie(cname,cvalue,exdays)/*cookie 的名称为 cname，cookie 的值为 cvalue*/
{
  var d = new Date();
  d.setTime(d.getTime()+(exdays*24*60*60*1000));
  var expires = "expires="+d.toGMTString();/*设置 cookie 的过期时间*/
  document.cookie = cname + "=" + cvalue + "; " + expires;
}

获取 cookie 值：
function getCookie(cname)
{
  var name = cname + "=";
  var ca = document.cookie.split(';');
  for(var i=0; i   {
    var c = ca[i].trim();/*去除前后空格*/
    if (c.indexOf(name)==0) return c.substring(name.length,c.length);
  }
  return "";
}
