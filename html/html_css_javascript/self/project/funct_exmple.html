简介开始
  1. !DOCTYPE html -> 声明为 HTML5 文档
  2. html -> 元素是 HTML 页面的根元素
  3. head -> 元素包含了文档的元（meta）数据，如[meta charset="utf-8"]定义网页编码格式为 utf-8
  4. title -> 元素描述了文档的标题，浏览器中工具栏标题、收藏夹标题、搜索引擎结果页面标题
  5. body -> 元素包含了可见的页面内容
  6. h1-h6 -> 元素定义一个大标题(小标题)
  7. p -> 元素定义一个段落
  8. br -> 换行
  9. 属性 -> 以名称/值对出现，如链接的地址在"href"属性中指定。属性值应该始终被包括在引号内。双引号是最常用的，不过使用单引号也没有问题（若属性值本身有双引号则必须使用单引号）
  10. class -> 为html元素定义一个或多个类名，类名从样式文件中引入
  11. id -> 定义元素的唯一id
  12. hr -> 定义水平线，用水平线隔开两段不同的内容，易于查看
  13. b -> 粗体文本
  14. em -> 着重文字，如变斜体
  15. i -> 斜体字
  16. small -> 小号字
  17. strong -> 加重语气，如变粗体字
  18. sub -> 下标字，
  19. sup -> 上标字，如x的平方
  20. ins -> 插入字
  21. del -> 删除字
  22. code -> 计算机输出代码
  23. kbd -> 键盘输入码
  24. tt -> 打印机文本
  25. samp -> 计算机代码样本
  26. var -> 变量
  27. pre -> 与格式文本，此处的格式和写入的格式一样，比如不需要加br直接换行，在浏览器中也可显示换行和空格等
  28. abbr -> 缩写
  29. acronym -> 首字母缩写
  30. address -> 地址
  31. bdo -> 文字方向，dir="rtl"为文字从右到左显示
  32. blockquote -> 长的引用，自动有一特定的块来描述从长段
  33. q -> 短的引用语，块引用，此时会自动在此文字段前后加入引号
  34. cite -> 引用、引证，此时会斜体显示文字
  35. dfn -> 定义一个定义项目
  36. base -> 定义所有链接的默认目标url
  37. link -> 文档与外部资源的关系，如css或js，通常链接到样式表
  38. script -> 客户端的脚本文件，如javascript，通常用于图像操作、表单验证、动态内容修改等

    html5的新属性:
    async -> 规定异步执行脚本
    charset -> 在脚本中使用的字符编码
    defer -> 当页面已完成解析后执行脚本
    src -> 外部脚本的url
    type -> 脚本的mime类型
    noscript : 若不支持scriopt则显示noscript部分的信息

  39. div -> 块级元素，可用于组合其他HTML元素的容器
  40. span -> 内联元素，可用作文本的容器
  41. iframe -> 通过使用框架，你可以在同一个浏览器窗口中显示不止一个页面

    src : 指向不同的网页
    height/width : 宽高
    frameborder : 是否显示边框

    如下例子：因为 a 标签的 target 属性是名为 iframe_a 的 iframe 框架，所以在点击链接时页面会显示在 iframe框架中
iframe src="table_exmple.html" name="iframe_a"> /iframe>
p> a href="./text_exmple.html" target="iframe_a">RUNOOB.COM /a> /p>

  42. 位与nbsp; -> 空格（由于&这个不能和nbsp一起，否则显示不出来）
  43. a位与#768; -> à ,其中&表示位与
  44. a位与#769; -> á ,其中&表示位与
  45. a位与#770; -> â ,其中&表示位与
  46. a位与#771; -> ã ,其中&表示位与
  47. 位与lt; ->小于号,<.
  48. 位与gt; -> 大于号,>.
  49. 位与amp; -> 和号,&.
  50. 位与quot; -> 引号,".
  51. 位与yen; -> 人民币,¥.
  52. 位与copy; -> 版权,©.
  53. 位与reg; -> 注册商标,®.
  54. 位与trade; -> 商标,™.
  55. 位与times; -> 乘号,×.
  56. 位与divide; -> 除号,÷.
  57. 位与sect; -> 小节,§.
  58. url -> scheme://host.domain:port/path/filename

    59. scheme - 定义因特网服务的类型。最常见的类型是 http/https/ftp等
    host - 定义域主机（http 的默认主机是 www）
    domain - 定义因特网域名，比如 runoob.com
    port - 定义主机上的端口号（http 的默认端口号是 80）
    path - 定义服务器上的路径（如果省略，则文档必须位于网站的根目录中）
    filename - 定义文档/资源的名称

    URL 编码使用 "%" 其后跟随两位的十六进制数来替换非 ASCII 字符
    URL 不能包含空格。URL 编码通常使用 + 来替换空格

链接功能开始
  1. a href -> 链接的目标地址，可以是本地也可以是网页
  2. target -> _top为不新建页面直接在本页面打开链接，即跳出框架;_blank为在新窗口打开链接
  3. a href="#xx" a id="xx" -> 从href处跳转到指定位置id处
  4. img -> html图像；图片链接

    border ->有无边框（0无边框，其他值为有边框）
    src -> 地址，本地或远程url都可以
    alt -> 替用，图像加载不出来时显示的提示信息
    width,herght -> 图片的长宽
    map -> 创建图片映射（其中area为图片中可点击的区域）

    map的使用:
    通过img得到原图，然后在原图img处定义一个usemap="#planetmap"属性。
    之后使用 map name="planetmap"来获取此图。在map里面定义其他的映射图与对应此图的位置，点击对应位置得到映射的图并跳转过去
    area shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm" area为图片中的可点击区域。shape为定义一个矩形框，coords为在图中坐标xy轴以及长宽，
    area shape="circle" coords="90,58,3" alt="Mercury" href="mercur.htm" 为定义一个圆形，coords为图中xy坐标和半径
    即(去除了尖括号):

    	图像映射的使用如下：
	img src="planets.gif" width="145" height="126" alt="Planets" usemap="#planetmap"
	map name="planetmap"
		area shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm"
		area shape="circle" coords="90,58,3" alt="Mercury" href="mercur.htm"
		area shape="circle" coords="124,58,8" alt="Venus" href="venus.htm"
	/map

表单功能开始(按钮、单复选框、文本、下拉等)
  1. form -> 定义供用户输入的表单
  2. input -> 定义输入域
  3. textarea -> 定义文本域 (一个多行的输入控件)，可实现文本编辑
  4. label -> 定义了input元素的标签，一般为输入标题
  5. fieldset -> 定义了一组相关的表单元素，并使用外框包含起来
  6. legend -> 定义了fieldset 元素的标题
  7. select -> 定义了下拉选项列表
  8. optgroup -> 定义选项组
  9. option -> 定义下拉列表中的选项
  10. button -> 定义一个点击按钮
  11. datalist -> 指定一个预先定义的输入控件选项列表
  12. keygen -> 定义了表单的密钥对生成器字段
  13. output -> 定义一个计算结果
  14. action -> 为提交时的保存位置，form中的属性
  15. method -> 提交方式post或get，form中的属性
  16. input type="text" -> 定义单行的输入字段，用户可在其中输入文本。默认宽度为 20 个字符
  17. input type="password" -> 输入密码，密文
  18. input type="checkbox" checked="checked" -> 复选框
  19. input type="radio" checked="checked" -> 单选框
  20. input type="submit" value="Send" -> 定义提交按钮。提交按钮会把表单数据发送到服务器
  21. file -> 定义输入字段和 "浏览"按钮，供文件上传
  22. hidden -> 定义隐藏的输入字段
  23. image -> 定义图像形式的提交按钮
  24. reset -> 定义重置按钮。重置按钮会清除表单中的所有数据
  25. selected="selected" -> 下拉时默认的开始选项
