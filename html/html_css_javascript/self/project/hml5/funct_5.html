canvas功能实现
  1. canvas -> 定义图形，比如图表和其他图像，您必须使用脚本来绘制图形，在画布上（Canvas）画一个红色矩形，渐变矩形，彩色矩形，和一些彩色的文字

    2. 以下为颜色、样式和阴影

  2. fillStyle -> 设置或返回用于填充绘画的颜色、渐变或模式
  3. strokeStyle -> 设置或返回用于笔触的颜色、渐变或模式
  4. shadowColor -> 设置或返回用于阴影的颜色
  5. shadowBlur -> 设置或返回用于阴影的模糊级别。
  6. shadowOffsetX -> 设置或返回阴影与形状的水平距离
  7. shadowOffsetY ->
    8. 设置或返回阴影与形状的垂直距离。
  8. createLinearGradient() -> 创建线性渐变（用在画布内容上）
  9. createPattern() -> 在指定的方向上重复指定的元素
  10. createRadialGradient() -> 创建放射状/环形的渐变（用在画布内容上）
  11. addColorStop() -> 规定渐变对象中的颜色和停止位置。

    12. 以下为线条样式

  12. lineCap -> 设置或返回线条的结束端点样式
  13. lineJoin -> 设置或返回两条线相交时，所创建的拐角类型
  14. lineWidth -> 设置或返回当前的线条宽度
  15. miterLimit -> 设置或返回最大斜接长度

    16. 以下为矩形

  16. rect -> 创建矩形
  17. fillRect -> 绘制"被填充"的矩形
  18. strokeRect -> 绘制矩形（无填充）
  19. clearRect -> 在给定的矩形内清除指定的像素。

    20. 以下为路径

  20. fill -> 填充当前绘图（路径）
  21. moveTo -> 把路径移动到画布中的指定点，不创建线条，线条的开始坐标
  22. lineTo -> 添加一个新点，然后在画布中创建从该点到最后指定点的线条，线条的结束坐标
  23. stroke -> 绘制已定义的路径，由moveto和lineto两点确定一条线，这里是绘线
  24. beginPath -> 起始一条路径，或重置当前路径
  25. closePath -> 创建从当前点回到起始点的路径
  26. clip -> 从原始画布剪切任意形状和尺寸的区域
  27. quadraticCurveTo -> 创建二次贝塞尔曲线
  28. bezierCurveTo -> 创建三次贝塞尔曲线
  29. arc -> 创建弧/曲线（用于创建圆形或部分圆）
  30. arcTo -> 创建两切线之间的弧/曲线
  31. isPointInPath -> 如果指定的点位于当前路径中，则返回 true，否则返回 false

    32. 以下为转换

  32. scale -> 缩放当前绘图至更大或更小
  33. rotate -> 旋转当前绘图
  34. translate -> 重新映射画布上的 (0,0) 位置
  35. transform -> 替换绘图的当前转换矩阵
  36. setTransform -> 将当前转换重置为单位矩阵。然后运行 transform

    37. 以下为文本

  37. font -> 设置或返回文本内容的当前字体属性
  38. textAlign -> 设置或返回文本内容的当前对齐方式
  39. textBaseline -> 设置或返回在绘制文本时使用的当前文本基线
  40. fillText -> 在画布上绘制"被填充的"文本，实心字体
  41. strokeText -> 在画布上绘制文本（无填充），空心字体
  42. measureText -> 返回包含指定文本宽度的对象

    43. 以下为图像绘制

  43. drawImage -> 向画布上绘制图像、画布或视频

    44. 以下为像素操作

  44. width -> 返回 ImageData 对象的宽度
  45. height -> 返回 ImageData 对象的高度
  46. data -> 返回一个对象，其包含指定的 ImageData 对象的图像数据
  47. createImageData -> 创建新的、空白的 ImageData 对象
  48. getImageData -> 返回 ImageData 对象，该对象为画布上指定的矩形复制像素数据
  49. putImageData -> 把图像数据（从指定的 ImageData 对象）放回画布上

    50. 以下为合成

  50. globalAlpha -> 设置或返回绘图的当前 alpha 或透明值
  51. globalCompositeOperation -> 设置或返回新图像如何绘制到已有的图像上

    52. 以下为其他功能

  52. save -> 保存当前环境的状态
  53. restore -> 返回之前保存过的路径状态和属性

内联svg

SVG 指可伸缩矢量图形;用于定义用于网络的基于矢量的图形;使用 XML 格式定义图形;图像在放大或改变尺寸的情况下其图形质量不会有损失;是万维网联盟的标准;可通过文本编辑器来创建和修改;可被搜索、索引、脚本化或压缩;可在任何的分辨率下被高质量地打印;可在图像质量不下降的情况下被放大; SVG 是一种使用XML描述2D图形的语言,而Canvas通过JavaScript来绘制2D图形
