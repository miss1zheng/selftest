简介

html：超文本标记语言，是一种用于创建网页的标准标记语言。文档后缀为 .html 或 .html 。

例如：
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>html</title>
</head>
<body>
  <p>html主体。</p>
</body>
</html>

解析：
<!DOCTYPE html> 声明为 HTML5 文档
<html> 元素是 HTML 页面的根元素
<head> 元素包含了文档的元（meta）数据，如 <meta charset="utf-8"> 定义网页编码格式为 utf-8。
<title> 元素描述了文档的标题
<body> 元素包含了可见的页面内容

注：在浏览器的页面上使用键盘上的 F12 按键开启调试模式，就可以看到组成标签。

标签：1.HTML 标签是由尖括号包围的关键词，比如 <html>；2.HTML 标签通常是成对出现的，比如 <b> 和 </b>；3.标签对中的第一个标签是开始标签，第二个标签是结束标签；4.开始和结束标签也被称为开放标签和闭合标签

HTML元素：1.包含了开始标签与结束标签，以开始标签起始，以结束标签终止；2.元素的内容是开始标签与结束标签之间的内容；3.某些 HTML 元素具有空内容（empty content），空元素在开始标签中进行关闭（以开始标签的结束而结束）；4.大多数 HTML 元素可拥有属性

HTML属性：1.HTML 元素可以设置属性;属性可以在元素中添加附加信息；2.属性一般描述于开始标签；3.属性总是以名称/值对的形式出现，比如：name="value"；4.属性值应该始终被包括在引号内，双引号是最常用的，不过使用单引号也没有问题。在某些个别的情况下，比如属性值本身就含有双引号，那么您必须使用单引号。5.适用于大多数元素内的属性有：class/id/style/title等。

html编辑器选择：1.VS Code；2.Sublime Text；3.source insight；4.Notepad++等。

<!DOCTYPE html>：声明为 HTML5 文档

必须位于 HTML5 文档中的第一行

html： HTML 页面的根元素

限定了文档的开始点和结束点

属性：
lang：支持的语言，如lang="en"，英语，中文为”zh"。

head：包含了文档的元（meta）数据

用于定义文档的头部，它是所有头部元素的容器。<head> 中的元素可以引用脚本、指示浏览器在哪里找到样式表、提供元信息等等。文档的头部描述了文档的各种属性和信息，包括文档的标题、在 Web 中的位置以及和其他文档的关系等。绝大多数文档头部包含的数据都不会真正作为内容显示给读者。

可包含title、style、base、link、meta、script等元素

body：包含了可见的页面内容

定义文档的主体。包含文档的所有内容，比如文本、超链接、图像、表格和列表等等。

title：描述了文档的标题

title作为元素时描述文档的标题，浏览器工具栏的标题、当网页添加到收藏夹时，显示在收藏夹中的标题、在搜索引擎结果页面的标题，而作为属性时描述了元素的额外信息 (作为工具条使用)

例如：
<!DOCTYPE html>
<html>
<head>
  <title>html</title>
</head>
<body>
 <p title="这个就是title属性的功能哟">请将鼠标移动到此处</p>
</body>
</html>
结果：

请将鼠标移动到此处

a：链接

设置超文本链接。超链接可以是一个字，一个词，或者一组词，也可以是一幅图像。点击这些内容来跳转到新的文档或者当前文档中的某个部分。

把鼠标指针移动到网页中的某个链接上时，箭头会变为一只小手。

默认链接在浏览器中形式为：1.未访问过的链接显示为蓝色字体并带有下划线；2.访问过的链接显示为紫色并带有下划线；3.点击链接时，链接显示为红色并带有下划线。

属性：
href：指定链接地址，如href="http://www.baidu.com/"（注意：请始终将正斜杠添加到子文件夹，即若最后不是.xx文件则需要在后面加正斜杠）；
target：定义被链接文档在何处显示，如target="_blank"即在新窗口打开，而target="_top"直接在本窗口打开；
id：创建HTML文档书签标记，跳转到文档指定位置（若为空则会跳转到页面最顶端），如id="this1"(此时href="#this1")可从href定义处点击跳转到id使用处（也可以直接使用href="xxx/xx.html#this1"即从另一个页面跳转到xx页面的this处）。

例如：
<a href="http://www.baidu.com" target="_blank">百度（www.baidu.com）</a>
结果：
百度（www.baidu.com）

img：图像

属性：
src：指定图像地址，源属性的值是图像的 URL 地址，URL 指存储图像的位置，可以是网页图链接//www.baidu.com/xxx/xx.png，也可以是本地链接/xxx/xx.gif；
width和height：指定图像大小尺寸，默认单位为像素；
alt：属性标识图像无法载入时的可替换的文本描述；
border：边框大小，为0表示没有边框
usemap：映射，配合map、area元素使用
draggable="true"：使元素可拖动
ondragstart：拖动对象
ondragover：放置位置
ondrop：进行放置

input：输入域

属性：
type：
1.type="text"为纯文本域；
2.password为密码输入；
3.radio单选按钮（配合value唯一值确定点击单一的input进行单选）；
4.checkbox复选（配合value一起）；
5.submit提交按钮（当用户单击确认按钮时，表单的内容会被传送到另一个文件，此文件定义在form里面的action属性里面）；

value：预输入值，开始的默认值。
name：提交时带有的键
checked：复选或单选框里预选勾上，默认选上。

autofocus：布尔值，规定在页面加载时，域自动地获得焦点。
formaction：描述表单提交的URL地址。会覆盖 <form> 元素中的action属性。 formaction 属性用于 type="submit" 和 type="image"。
formenctype：描述了表单提交到服务器的数据编码 (只对form表单中 method="post" 表单)
formmethod：表单提交的方式。覆盖了 <form> 元素的 method 属性。该属性可以与 type="submit" 和 type="image" 配合使用。
formnovalidate：布尔值，描述了 <input> 元素在表单提交时无需被验证，会覆盖 <form> 元素的novalidate属性。注意: formnovalidate 属性与type="submit一起使用。
formtarget：指定一个名称或一个关键字来指明表单提交数据接收后的展示，覆盖 <form>元素的target属性。注意: formtarget 属性与type="submit" 和 type="image"配合使用。
height和width：h规定用于 image 类型的 <input> 标签的图像高度和宽度。
list：输入域的 datalist。datalist 是输入域的选项列表。
min和max：min、max 和 step 属性用于为包含数字或日期的 input 类型规定限定（约束）。
multiple：布尔值，规定 元素中可选择多个值。适用于type="email" 和 file类型。
pattern：描述了一个正则表达式用于验证 <input> 元素的值。
placeholder：提供一种提示（hint），描述输入域所期待的值。简短的提示在用户输入值前会显示在输入域上。
required：布尔值，规定必须在提交之前填写输入域（不能为空）。
step：为输入域规定合法的数字间隔。如果 step="3"，则合法的数是 -3,0,3,6 等。step 属性可以与 max 和 min 属性创建一个区域值。

canvas：定义图形

比如图表和其他图像。该标签基于 JavaScript 的绘图 API。<canvas> 标签只是图形容器，必须使用脚本（通常是JavaScript）来绘制图形。默认情况下 <canvas> 元素没有边框和内容，但是可以通过style添加边框。

SVG：可伸缩矢量图形，用于网络的基于矢量的图形，使用 XML 格式定义图形，在放大或改变尺寸的情况下其图形质量不会有损失。
SVG 与 Canvas两者间的区别：SVG 是一种使用 XML 描述 2D 图形的语言；Canvas 通过 JavaScript 来绘制 2D 图形。

例如：
<canvas id="myCanvas" width="200" height="100" style="border:1px solid #c3c3c3;">
您的浏览器不支持 HTML5 canvas 标签。
</canvas>
<script>
  var c=document.getElementById("myCanvas"); //找到 <canvas> 元素
  var ctx=c.getContext("2d"); //创建 context 对象
  ctx.fillStyle="#FF0000"; //填充
  ctx.fillRect(0,0,150,75); //绘制的矩形
  ctx.moveTo(0,0); //线条开始坐标
  ctx.lineTo(200,100); //线条结束坐标
  ctx.beginPath();
  ctx.arc(95,50,40,0,2*Math.PI); //绘制圆形
  ctx.stroke();
</script>
