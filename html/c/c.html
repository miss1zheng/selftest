第一章编程练习

1.你刚被MacroMuscle有限公司聘用。该公司准备进入欧洲市场，需要一个把英寸单位转换为厘米单位（1 英寸=2.54 厘米）的程序。该程序要提示用户输入英寸值。你的任务是定义程序目标和设计程序（编程过程的第1步和第2步）。

** ** #define fix_value 2.54
int main()
{
  float data=0;
  printf("please you input a value:");
  scanf("%f",&data);
  printf("%f = %f\n",data,data*fix_value);
  system("pause");
  return 0;
}

第二章编程练习

1.编写一个程序，调用一次 printf()函数，把你的姓名打印在一行。再调用一次 printf()函数，把你的姓名分别打印在两行。然后，再调用两次printf()函数，把你的姓名打印在一行。输出应如下所示（当然要把示例的内容换成你的姓名）：
Gustav Mahler ——第1次
Gustav ——第2次
Mahler ——第2次
Gustav Mahler ——第3、4次

** ** #define fix_value 2.54
int main()
{
  printf("%s\n","Gustav Mahler");
  printf("%s\n","Gustav\nMahler");
  printf("%s ","Gustav");
  printf("%s\n","Mahler");
  system("pause");
  return 0;
}

2.编写一个程序，打印你的姓名和地址。

** ** #define fix_value 2.54
int main()
{
  char data[20],addr[50];
  printf("please input your name and addr:");
  scanf("%s%s",data,addr);
  printf("your name is:%s,and addr is:%s",data,addr);
  system("pause");
  return 0;
}
注意：scanf格式化时%s是按空格作为分隔符的，也就是输入时若有空格，则空格后的数据就作为下一个数据
